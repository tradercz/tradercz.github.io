7月8日BTC行情：主流退位，山寨登场

发表于 2020-07-08 分类于 行情分析

过去24小时，整个市场的交易量559亿美金，比前一天减少了约200亿，现在是北京时间7月8日的上午08点29分，BTC即时价格9238USDT，下面是具体分析。

1）昨天BTC最高9379，最低9203，交易量有所减少，最近的行情实在无聊，该跌不跌，该涨不涨，这其实是市场缺少人气的表现。大结构上，仍属于8600-10500区间，不跌破不突破视为大结构不变。小结构上，昨天尝试突破失败，在接近9400时受到了30日线、60日线以及4小时下降趋势线的共同压制，但由于目前已经在4小时的MA30和MA60上方运行了一段时间，短期还是强势行情后的震荡整理。上方压力9370-9400区间，下方支撑9150-9200区间。总结一下，中期趋势下跌，短期还是强势行情，后面重点关注9400附近的压力位，突破后的目标是9600以及9800，突破9800，下跌趋势结束，后市会再次测试10500这个长期压力位。当然，如果将多军看作进攻方，在9400、9600和9800等每一个关键位置都有可能失败，这点是大家需要注意的。

2）如果你研究过交易心理学，你就会发现其内容反反复复就那么几句，然而就是这么简单的几句，却难倒了千千万万个小韭菜，如何破？要么自己去历经，要么经常将交易心理放在手边，耳濡目染。其中最有效的还是自己去经历，但这可能需要付出很大的代价，有些人甚至会付出一生的代价，所以为了追求性价比，咱们可以选择第二种方法——我不时的唠叨给你听。今天的话题是成熟的交易者与不成熟的交易者之间的区别。我认为，主要区别如下：成熟的交易者重视风险，不成熟的交易者只看到机会；成熟的交易者害怕追高，不成熟的交易者害怕踏空；成熟的交易者只做自己看的懂的，不成熟的交易者喜欢做过山车……这也能解释为何在牛市中新手往往赚的更多，新手看不到风险，初生牛犊不怕虎，在牛市中反而更容易抓住机会（牛市机会多），但是呢？在趋势转变之后，新手往往无法及时调整策略，还沉醉在牛市的幻想中，很容易将之前赚的赔进去，甚至赔更多。你看17年18年两年，坐过山车的人就非常多，有的甚至到现在都没有解套。至此，你能否从上面的话中得出一些感悟呢？

3）最近主流币被山寨币抢尽了风头，其实主流币和山寨币并没有本质上的区别，山寨币市值大了以后，系统性风险就会减小，交易的人也会越来越多，这时候山寨币就变成了主流币。

那么有哪些山寨币最有可能成为主流币呢？或者有哪些是新的主流币呢？

7月7日BTC行情：行情反转了吗？

发表于 2020-07-07 分类于 行情分析

过去24小时，整个市场的交易量734亿美金，比前一天增加了200多亿，现在是北京时间7月7日的上午08点30分，BTC即时价格9350USDT，下面是具体分析。

1）终于，BTC的交易量上来了，虽然距离顶峰时期的水平还差很远，但增量就代表了大家交易热情的递增。行情方面，已经到了咱们强调了很久的9300-9400附近，这里不仅有筹码交投高峰，还是30、60日线的防守位，目前这两条线已经拐头向下，对BTC起着决定性的压制作用，也决定着中期趋势的方向。那么行情反转了吗？很明显还没有。如果要反转，需要满足什么样的条件呢？首先要站上30、60日线，然后再突破9800，只有满足了这两个条件，才能确认行情已经反转，否则随时都会受到圆弧顶的威胁。当然，你要问成功反转很困难吗？如果放在之前，是非常困难的，但现在有所不同了，主要是外部金融市场的做多热情高涨，如果股市中的一小部分资金能进入币市，那也会导致一波小高潮的发生。总结一下，大趋势仍然受空头压制，但操作上并不建议做空，谁知道A股的疯牛情绪会不会传染到币市中来呢？

2）就在写这篇文章时，有人私信问我今天涨还是跌，说实话，我不清楚，不仅今天不清楚，明天也大概率不清楚，除了极少数高确定性的情况之外，一年365天中，大部分的日子我都不知道涨还是跌。那么不知道涨跌是不是就不能盈利呢？真相是不知道涨跌并不能和不能盈利划等号，原因我之前讲过，每次开单前都要计算盈亏比，只要盈亏比合适，哪怕你只有10％的胜率，结果是一样赚钱的，反过来说，哪怕你有90%的胜率，只要盈亏比不合适，你仍然赚不到钱。如果你不太理解，就通过网络研究一下，只要有初中的数学基础，都是可以搞懂的。

当前行情见顶了吗？如何判断行情见顶？

发表于 2020-07-07 分类于 行情分析

这是一篇6月11日的旧文，依然使用于现在的行情，故发了上来。

好久没有细谈行情了，就天来说一下。

在公开版的分析中，我没有明确的看多还是看空，但在咱们内部，我会给出我的多空观点的。

先给出结论：目前看多更多一些。

理由：

1、从K线图上，并未发现明显的出货迹象，只是这里有个类似日线双顶的形态比较容易迷惑人，但既然出货迹象不明显，那咱就暂且不要看的太空。

2、从最近的走势上，多次上冲后马上插针回来，给人一种多头无力的感觉，但5月24日以及6月2日的下跌后很快就会小碎步涨上来这一现象来看，主力洗盘的嫌疑更重。

3、从散户情绪上来看，以往每一次的顶部都是伴随着散户的疯狂而到来，你看现在散户的情绪，始终呈现一个中性的状态，即使上次突破了10000点，也未见散户疯狂的买入，甚至有很多散户都很迷惑、怀疑甚至不看好后市，之所以发生此现象，我能想到的原因就是目前的行情很可能没有走完，散户不疯狂，顶部大概率就还没有到来。

4、从最近转入转出交易所的BTC的数量来看，每当顶部到来时，转入交易所的BTC就会变多，但现在的情况是一直从交易所向外面转币，说明无法形成潜在的抛压。

5、从Bitmex永续合约的费率来看，每次行情行情到顶之前，费率总会达到0.1％，最高的能达到0.3以上，而现在的费率仅有0.01％，这点费率对于多头来说能轻松承担，多头做多的意愿就不会减弱，而空头做空的意愿也不会增强。
