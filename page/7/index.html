1月5日比特币行情：不跌破这里，就说明回调结束了

发表于 2021-01-05 分类于 行情分析

过去24小时，整个市场的交易量1743亿美金，比前一天减少约1400亿，现在是北京时间1月5日的上午08点39分，BTC即时价格32760USDT，下面是具体分析。

1）昨天BTC带领众币回调，对上升趋势没有造成任何破坏，甩掉一批人后，大概率会继续上行。结构方面，目前已经突破了34778以来的下跌趋势线，如果后面回踩确认有效（不跌破32150，小时线MA30这条线），那么就说明回调结束了。上方压力33000、33500点。另外，经过昨天的一跌，各大交易所永续合约费率大幅下降，也说明洗盘很有效果。总结一下，只要不跌破32150，就说明回调结束了。

2）很多同学在昨天的快速下跌中损失惨重，今天就再谈谈风险控制吧。一、哪怕你再喜欢赌，也不要拿你所有的仓位做合约，最好是9份现货1份合约，这样即使被爆，也得爆10次，况且很多人被爆一次就长记性了。二、当我们发现方向错误时，就要勇于止损，被套后再预测行情就已经晚了，更不要抱有侥幸心理去补仓，要学会放下，很多时候，补仓还不如做对冲。三、拿一笔输的起的钱来交易，哪怕赔光了，也不会影响你的生活。四、仓位管理很重要，永远不梭哈。五，不要用10倍以上杠杆，低杠杆交易很舒服，还延年益寿。六、在行情上涨时，有两种加仓方式，一种是突破后加仓，这叫趋势性加仓，一种是上涨趋势中的回踩加仓，这叫结构性加仓，我推荐大家用结构性加仓的方式。七，上面提到的的加仓又滚仓，滚仓最最要的是风险控制，很多同学滚到最后都被爆了。八、上涨的时候不要来问我还能买吗，下跌的时候才要考虑去买。暂时就这么多了，想到再补充。

1）BTC已经突破了29000点，上涨趋势继续强势，没有衰竭迹象，没有见顶形态，没有加速，后市大概率会继续上涨，但是，需要大家注意的是，目前比特币价格已经大幅偏离30日均线，这意味着市场有回调的动能，回调目标26000、24000。当然，我们并不知道回调何时发生以及会回调到哪里，之所以提醒你可能会回调是为了让你更加深刻的理解这个市场。后市走势方面，如果没有放量没有加速没有冲高回落，还是像最近这些天一天只是小幅度回调，那么后市将会继续上攻，30000点指日可待。总结一下，在其他信号出现之前，上涨趋势将持续下去。

12月27日比特币行情：强势行情

发表于 2020-12-27 分类于 行情分析

过去24小时，整个市场的交易量1888亿美金，几乎与前一天持平，现在是北京时间12月27日的上午07点58分，BTC即时价格26458USDT，下面是具体分析。

1）连续三天的放量拉升说明BTC的上涨趋势很强，在出现见顶信号之前，不建议猜顶。结构上，上方一望无际，下方支撑25500、25000、24000点。链上数据方面，昨天提到的旷工抛售没有再继续，交易所转入转出同时在减少，可能是圣诞假期所导致的。但从昨晚的拉升看，仍然有资金在不停的进场，目测是从美金渠道直接入场的。总结一下，强势行情，即使有短暂的下跌也大概率是对行情的修复所致的。

2）根据我个人的经验，有两种习惯会对我们的交易产生不利影响，一是频繁的预测行情，二是频繁的看别人预测行情。频繁的预测行情，是对自己的策略没有把握的表现，上午觉得会涨，下午觉得会跌，总是害怕后市走势与自己预测的不一致，从而导致操作上的反反复复。而频繁的看别人预测行情，只不过是为了给自己所想找证据罢了，做多的喜欢看多方言论，做空的喜欢看空方言论，彷佛只有找到自己的队伍才不会被这个世界所抛弃。还有一部分人，他们没有自己的多空观点，看到这个大V看多就开多，看到那么大V看空就开空，如此变幻莫测的操作，仿佛只有世界顶级的交易员能与一战似的。那么正确的做法是怎样的呢？我之前说过，那就是一旦开仓后就尽量不要预测行情了，只要属于你的平仓信号没有出现就一直拿着。比如在一段上升趋势中，我们的交易信号是跌破上升趋势线减仓，那我们盯着这个信号就行了，只要它不出现，就果断的拿着，不要去想，更不要去听有可能影响自己操作的言论。总之大原则是不要因为你临时的想法或者别人的一句话而改变自己的交易，要时刻记住，预测是预测，交易是交易。
