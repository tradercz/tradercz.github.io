8月16日比特币行情：向上突破只是时间问题

发表于 2020-08-16 分类于 行情分析

过去24小时，整个市场的交易量1031亿美金，几乎与前一天持平，现在是北京时间8月16日的上午08点54分，BTC即时价格11833USDT，下面是具体分析。

1）目前BTC处于三角震荡，上方12000，下方11500，变盘时间就在下周了，目前还是更倾向于向上突破。细节方面，总体来说没太大问题，可能是大家都在等突破，再加上对未来一年两牛市的看好，惜售情绪重，因此BTC抛压并不严重，从这点上来说BTC并没有大幅下跌的需求。然而这并不意味着就不会跌了，由于有高倍杠杆的存在，那对于主力来说，狙杀高杠杆就是一项很重要的任务，就算今天下跌500点也不要意外，最重要的是，牛市中的每一次下跌都能很快收复，这往往也是我们入场的机会，因此，低倍杠杆是长期盈利的基础。

2）想象两个场景，一是马云比你挣钱多，你并不会有什么感觉；二是你老婆的妹妹的老公比你挣钱多，你就有点不爽了。这个不爽就是嫉妒心。人都是有嫉妒心的，而且在远古时期嫉妒心是好事，有嫉妒心的人更容易活下来，那么他们的基因就更容易传下去，久而久之，大家都带有或多或少的嫉妒心。这里的嫉妒心其实就是行为经济学里的参考点依赖，是偏好中偏差的一种。参考点依赖对我们交易有什么指导意义呢？我认为最重要的指导意义是能够在关键时刻帮你选择要卖的标的。假如你前段时间各用10000块钱买入了A和B两个币，到了今天，A盈利，B亏损，现在你需要一笔资金，你必须在A和B之间选择卖出一个才行，你会卖出哪个？这里的10000块就是参考点，当你盈利时，由于风险厌恶，你更容易锁定利润，但当你亏损时，也就是低于10000这个参考点时，你就更愿意去赌博，因此，大部分人都会选择卖出A。我要告诉你的是，这样是不对的，你应该卖出亏损的B而保留盈利的A，而且根据我的经验，这条规则在币圈同样适用。币圈有很多人，盈利了不敢继续拿着，一看回调了就马上卖出，从而错过了后面更大的行情，而亏损后索性就不管了，还名曰“死拿”，结果是腰斩再腰斩，哪怕后面涨个10倍也无法抹平亏损。想一想，你有过这种经历吗？

1）BTC没太多可说的，继续看什么时候能够突破就好，上方压力12000、12123点，下方支撑11850点。ETH方面，很强势，突破366并经过回踩确认后，ETH算是彻底走出了2年之久的底部区间，2017-2018年的每个高点都会成为接下来要征服的目标，比如510、550、630、830……至于最终能到达什么样的高度，不清楚，走一步看一步吧。需要注意的是，不要看了本篇文章就头脑发热去梭哈ETH，不考虑风险和波动的交易都是赌博。

2）经常交易的人一定会遇到怎么做怎么错的情况，也就是说，在某段时间内，无论你怎么做，都是错的。你做多，亏损，做空，继续亏损，更换标的，还是亏损，你总觉得市场与你作作对，甚至全世界都在针对你。遇到此类情况，如何做才科学呢？首先要端正态度，将市场和你作对这一想法删除，市场没有和任何人作对，一切都是人的错，是人想和市场作对。接着仔细思考一下，你在这段时间内有没有盲目开单，是想着征服某个标的？还是想和某人赌气？还是与市场赌气？又或者说害怕错过大行情？甚至是单纯的想抓住每一次机会？很多时候我们越做越错，都是心态乱了，心乱则谋去，将心态摆正，我们才能谈盈利。最后，将焦点放在我们的策略上来，当前行情是否适合用该策略，如果不合适，是不是要改进了？当然，上述是解决这一问题的通用方案，但我亲身经历告诉我，最有效的方法并不是上面提到的，而是停止交易。不过在停止交易之前，我还是建议你看看上面的通用办法，若你心态有问题，是很难成功停止交易的。

2）昨天下跌时有VIP会员问到关于行情的问题，下面是我的当时的看法及回复——

这波下跌由YAM引起，但我们不讨论该事件本身，而是谈谈由该事件引出的一个名词——突发性利空。所谓突发性利空，往往发生在一片其乐融融的牛市中，此时大家的做多热情很足，都认为市场正朝着好的一方发展，突发性利空导致市场快速下跌，这是人在遇到突发状况时的一种正常表现，当人们对事件进行更详细的了解之后，恐惧情绪逐渐消散，随后的市场大都会很快恢复如初，因此，突发性利空往往是做多（建仓）的机会。当然，如果突发性利空大到足以影响到整个市场的基本面时，那就可能改变牛市的走向，这时候你再去做多就危险了，不过越是大的利空发生的概率越低，我们平时见到的利空都是一般性利空，大都是可以做多的。除此之外，大家也要明白，无论何时，无论市场看起来多么的健康，也都有可能随时发生意外，因此风险控制是大家在任何时候都要首先考虑到的。

1）昨天黄金跳水6个点，6个点对于黄金这种产品来说就是大瀑布，创下了7年来单日最大跌幅。BTC也受压下行，最低跌至11125点。目前行情如何了，11000上方的结构依然未破，第一个关键支撑就是11000，破了就看10500，这里也是很多人准备接货的地方。当然，破不破咱也不知道，单从目前来看，11000-11500有抄底盘，力度还行，但即使能够企稳，也不要想着很快就能上去，昨天这波下跌力度还是不小的，放量下跌，是最值得大家注意的地方，后市就看11000-11500之间震荡吧。另外要说的是，其实这波下跌BTC并没有跌多少，结构都没有破，但一些人的账面财富缩水很多，这说明要么杠杆太高，要么大的仓位没有在BTC上，而那些小币和庄控币的合约，就是散户收割机。

8月11日比特币行情：走势未变

发表于 2020-08-11 分类于 行情分析

过去24小时，整个市场的交易量1000亿美金，比前一天增加了260亿，现在是北京时间8月11日的上午09点01分，BTC即时价格11876USDT，下面是具体分析。

1）昨天的拉升并未改变震荡格局，大结构仍旧是10500-12123。小结构已经压缩至11800-12123区间，当区间被压缩的越来越小时，就是变盘的时候，咱们耐心等待即可。值得一说的是，每当行情进入震荡，就会有很多的人不耐烦，其实对于当前的行情，我倒觉得大家应该开心才对。你现在回头去看看去年6月那波行情，BTC站上10000点后的短短5天就冲到了13970，结果是牛市很快便没了。再看这次，今天已经是站上10000点的16天，虽然迟迟无法突破，但也没有下来，要知道在这里震荡的越久，突破后就越有可能成为强支撑，即使暂时不向上突破，那也比来的快去的快的牛市对市场伤害要小的多，因此，耐心等待，说不定很快就柳暗花明。

2）投资与交易有什么区别？先看我给他们下的定义，投资是用一定数量的钱做基础，通过时间换空间，以达到财富增值的目的。交易是用一定数量的钱做基础，通过空间换时间，以达到财富增值的目的。有什么区别？投资是付出时间，甚至用长达数年的时间来换取财富，而交易追求的是更快更多，通过多次交易、杠杆等途径实现财富的快速增值。举个例子，像学习、锻炼身体、长期拿着一家公司的股票就是投资，而交易更好理解，只要你发生买入和卖出行为，都是交易。当然，上面不是重点，今天的重点是要告诉你我们在交易时一定要有投资思维。怎么理解呢？在你还不懂时，能不能先潜下心来系统的学习？在你频繁交易却不赚钱时，能不能将交易频率降下来，将短线换成中长线？在你买入的标的没有涨时，能不能耐心等待而不是频繁更换标的？在你心烦气躁时，能不能停下来暂时不交易？要知道，很多时候你交易不成功，并不是技术不行，而是心态有问题，用投资思维指导交易，我看行。

8月10日比特币行情：警惕插针到11500的可能

发表于 2020-08-10 分类于 行情分析

过去24小时，整个市场的交易量734亿美金，几乎和前一天持平，现在是北京时间8月10日的上午08点52分，BTC即时价格11735USDT，下面是具体分析。

1）BTC还是没有突破，仍然是震荡行情，上看12000，下方10500，小结构上方11800和11900，下方11500和11300，1小时有回调的需求，再加上CME下方留有缺口，跌到11500附近即可补缺，因此，如果能跌到11500而不破，则是短线多单进场的机会，当然，这只是一种理想走势。总之，短期上要警惕插针到11500点的可能，大结构继续看震荡。

2）在8月8日的文章中，咱们介绍了过度自信对生活以及投资的影响，今天咱们来谈谈造成过度自信的原因。首先，我们都有一个偏差，叫自我归因偏差，说的是好的结果出现了，你觉得是自己的能力强；坏的结果出现了，你就觉得是自己运气不好。必须做交易，盈利了你会认为这是自己能力所得，而亏损时你就更可能的将原因归结到项目方、分析师身上。其次，我们都有证实偏差，与锚定效应有点像，就是说你一旦有一个观点，你可能会主动去寻找或者只记得那些跟你观点一致的证据。若你是多头，你总是在找那些有利于多头的数据或新闻，多头喜欢看多头的文章就是这个原因。再次，人都是有控制幻觉的，就是说本来无法控制的东西，你却觉得你可以控制。那些在交易中赚过大钱的人，更容易觉得自己可以控制市场，也就更容易吃大亏。最后，导致容易过度的另一个重要因素是人特别喜欢做因果关系的推理。怎么理解呢？就是说一件事情发生了，人们总是喜欢将那些不是原因的归为这件事发生的原因。比如某个交易所突然宕机了，之后币市就发生了大跌，大家就会认为宕机是大跌的原因。当然我们是无法确定宕机是否就是导致大跌的原因的，但你先将它看成原因后，就更容易主观化，之后在遇到同样的是事情时，你就会更容易确信后市一定会大跌，从而导致大的失误。

过去24小时，整个市场的交易量730亿美金，比前一天减少了约150亿，现在是北京时间8月9日的上午09点07分，BTC即时价格11738USDT，下面是具体分析。

1）BTC依然是震荡行情，上方压力位11900、12000，下方支撑11500、11300、11000，量能没有放大，没太多可说的，后市既然看震荡后会继续创新高。ETH还是很强势，在390-400区间震荡后有较大概率继续拉升。其他方面，咱们在8月5、6日两天提到的山寨币和新兴主流币会迎来第二春的观点正在变为现实，山寨币和新兴主流币之后，老牌主流币仍有机会，记住版块是会轮动的，你的主流币没有涨，请耐心等待。

2）交流群中经常能看到有人突然的喊道“涨了”、“跌了”的，当你打开K线去看时，发现实际波动还不到1个点，那为什么很多人很在意这不到1个点的波动呢？我认为原因有二。一是杠杆加的太高了，1个点放大几十上百倍，也足以让你相信暴涨暴跌的发生，这时候你就要适当降低杠杆了，否则很容易爆仓，正如今天币安上LINK空头爆仓，很多时候，爆仓如呼吸一样简单。二是你太在意几分钟内的走势了，你关注的周期越短，你策略改动的概率就会越大，你的准确度就会越低。如果你不是做超短线（8点买9点卖的那种），完全不必在意短时间内的涨跌。不看分钟级的走势，那看什么？如果你做日内交易，就是早上买晚上买的那种，重点看小时级别，而分钟级参考一下就行。如果你做的是3-5天的短线，那就重点看4小时周期。如果你做的是数周或者数月，那就要看日线和周线了。总之，遇事不要慌，先想一想你做什么周期的再说。

8月8日比特币行情：没有证据表明回踩已经完成

发表于 2020-08-08 分类于 行情分析

过去24小时，整个市场的交易量988亿美金，比前一天增加了约100亿，现在是北京时间8月8日的上午08点42分，BTC即时价格11586USDT，下面是具体分析。

1）昨天最高11909，最低11322，上下波动580点，量能方面，昨晚10点小时线有放量，不过总体量能并不大，可以理解为12000突破失败后的正常回踩，后市将继续走震荡行情。下方支撑11300、11000，上方压力11900、12000。短期走势方面，从昨天中午12点开始，4小时呈阶梯性放量，反弹无量，这种情形对多头不利，虽然我们还不清楚后市会不会二次回踩11300，但若真的回踩了，大家要做好防范。总结一下，大的上升趋势仍在，但短期仍有下探的可能，大家可以先关注11500点（4小时MA30附近）的得失。

2）咱们继续讲行为经济学，今天的话题是过度自信，你看到这个词马上就能理解它的意思，就是高估自己能力的错误。其实我们人类多多少少都会过度自信，比如男生大多都对自己的智商和开车技术过度自信，女生大多都对自己的颜值过度自信，特别是单身的人，为什么呢？因为单身的人有找配偶的需求啊，在人类长期的繁衍中，不自信的就会被淘汰掉，久而久之剩下的都是过度自信的。另外，在现代社会，那些能够创业成功的人也大都过度自信。这些都是过度自信的好处。那坏处呢？容易让人忽略风险，这点在投资中表现特别明显，那些过度自信的人喜欢频繁交易，但数据表明，对于大多数人而言，频繁交易并不能帮助你盈利更多，哪怕是专业人士也是如此。你可以留意一下那些大盘指数基金的数据，一般来说，换手率高的收益都比不上换手率低的，虽然每年只差1个多点，但要知道股神巴菲特也只是每年平均比大盘多5个点而已，单单两只基金就相差1个多单，差距已经很大了。另外，股评专家、分析师也会过度自信，为什么呢？因为不自信的都被淘汰了，反而是那些死多和死空有市场，因为这两类分析师都有忠实的粉丝，看空的人喜欢空头言论，看多的人喜欢多头言论，大家每天只能靠读他们的分析才能够支撑下去，因此，奉劝大家不要盲目相信分析师，特别是币圈的分析师。现在你知道了，在生活中，过度自信的好处更多一些，但在投资中，过度自信的坏处却更多，有何区别呢？生活中很多时候你是可以控制结果的，比如找配偶，你是喜欢乐观向上的还是郁郁寡欢的？当然是乐观向上的啊，而过度自信会使人更加乐观，你就更可能成功的找到配偶。而在投资中，你是控制不了结果的，那怕你有再大的资金，也总有控制不了市场的那一天，但大部分人都认为自己会是赚钱的那一个，这怎么可能呢？二八定律忘记了吗？投资并不是你过度自信就行了。因此，过度自信是好是坏，关键要看事情的结果能不能被你所控制。还有，那些还没结婚的女生，如果你男朋友在做投资（交易），一定要看好他哦，因为他更容易过度自信。好了，今天就说这么多，有时间咱们再接着讲都有哪些原因可以导致过度自信这一问题。
