11月17日比特币行情：这次17000真的不远了

发表于 2020-11-17 分类于 行情分析

过去24小时，整个市场的交易量2608亿美金，比前一天增加了1700多亿，现在是北京时间11月17日的上午08点23分，BTC即时价格15760USDT，下面是具体分析。

1）眼看着行情就要破位时，BTC再次拉升，目前已经远离了危险区，行情怎么看？上升趋势将持续下去，下一目标位就是2108年1月高点17176点。不过目前各项指标已经相互矛盾，周线上各种超买预示着后市有回调的可能，而且力度不会小，但日线上走的又很稳，完全看不出见顶迹象，再加上灰度等机构的助攻，使得BTC很难停下脚步，在这种情况下，我认为还是以当前的盘面为准，继续看涨，至于能走到哪里，不清楚，保持敬畏吧。总结一下，短期多头趋势再次形成，下一目标位17176点。

2）最近大家问的最多的一个问题就是是不是卖出某个币了，大家之所以有这样的疑问，大多是因为已经通过这些币取得了一定的收益，想要落袋为安。我对这一问题的看法是，如果你做的是短线合约，那么在每一个区间顶部或压力位附近都是卖出的点位，如果你做的是长线，尽量不要卖。行为金融学里有个过度交易理论，说的是相对于理性的交易频率而言，绝大多数人的交易都是过度的。而且大量案例证明，同一个标的，频繁交易并不比长期持有赚的更多，甚至会更少。大家之所以容易过度交易，与过度自信有很大的原因。所谓的过度自信，说的是投资者更容易相信自己掌握了充分的信息，可以进行交易了。而实际上，这些信息不足以支撑任交易。过度自信程度更高的投资者交易会更频繁，但由于交易成本存在，就会使得收益更差。除了过度自信外，过度交易还和长时间盯盘、不想看到自己的收益缩水有很大关系。当你看到自己的收益忽上忽下时，心里就会很难受，为了减少这种难受程度，在后面的行情中一旦遇到下跌就想逃亡，遇到上涨就想止盈。你现在知道了，为了获得更多的收益，尽量不要频繁操作，但还是不知道何时卖出。我的经验是，除非确认了顶部的存在，否则一直拿着。至于如何判断顶部，这就属于技术性问题了，比心理问题容易解决的多，你稍微学习一点技术知识就能试着去辨别，哪怕错了，也比你在每一次回调时都卖出要好。另外，如果你是VIP成员，到时候我也会凭借我的经验提醒到你。当然，不要过度交易并不是让你死拿，在该止盈或止损的时候，同样需要做到坚决和果断。

11月16日比特币行情：趋势正转向空头但不可过分看空

发表于 2020-11-16 分类于 行情分析

过去24小时，整个市场的交易量920亿美金，比前一天减少了约100亿，现在是北京时间11月16日的上午08点15分，BTC即时价格15912USDT，下面是具体分析。

1）连续两天的缩量回调，使BTC跌至16000点之下，不过昨天并没有跌破前一天的低点，仍是15800之上的震荡行情，下方支撑15800，上方压力16200。短期走势上，已经跌破了4小时MA30，若日内无法成功站上这条线，则趋势将有利于空方，但小时线上看15750-15800区间还是有不小的支撑力度的，只要不跌破15750，则不必过分看空。另外，CME最新缺口在16075-16325之间，若要回补，现货得涨至16180附近。总结一下，短期有利于空头，但在跌破15750之间，不要过分看空。还有，不要过分看空还有另一个原因，那就是灰度又要上班了，如果继续买买买，则可能就很难继续下跌了。

2）在我们投资的过程中经常喜欢做的一件事是：咨询他人的建议。 比如遇到一个比较懂或有过赚钱经验的朋友，总喜欢问问他最近买了哪些币，什么币有潜力等等。很多时候，你的投资建议就是根据朋友的建议做出的。这么做会有什么问题呢？会有利益错位的问题。什么是利益错位呢？那就是对方的利益与你的利益不一致，主要分为两种情况，一种是无论建议对你有没有好处，对对方都没有坏处，另一种是无论建议对你有没有好吃，对对方都有好处。第一种多发生在朋友之间，比如你买的币被套了，去问朋友建议，朋友有可能只是处于安慰你说没事，还会涨回来的等等。他这样说，可能只是想让你开心而已（对对方没坏处）。第二种多发生在网友之间，他推荐你买入一种币，可能只是出于利益的原因了，如你买入后他就有提成，又或者他在你买之前已经买了大量的币。最常见的是BCH、EOS等大V会告诉你买买买，即使被套了也会让你一直拿着。如果你买入的是BCH、EOS等主流币还算不错，毕竟这些币在牛市是非常有潜力的，如果是让你买一些不知名的小币的，一定要警惕，说不定给建议的人就是背后的庄。还有一种情况是只推荐你在某一个交易所交易，这也可能是出于利益的原因。当然，这并不是说不让你听从别人的建议了，毕竟这是我们反思和进步的重要方式。我只是在告诉你，不要简单地根据对方的建议采取行动，你应该判断对方的建议是他的利益有没有关系，如果有，是什么关系。另外，如果你要听从建议，也最好搞清楚背后的逻辑，不能单纯地因为某个大V看好就买入，你应该有自己的思考，在你还不懂时，千万不要下重注。

2）咱们来算一道数学题，如果你连续十年的投资收益是这样的：第一年赚20%，第二年亏15%；第三年赚20%，第四年亏15%；第五年又赚20%，第六年再亏15%……以此类推。那么这十年下来你总共的投资回报有多少呢？聪明的你很快就能算出是10.4%，如果平均到每一年，差不多1%，你可能会惊讶收益率怎么这么低，但这就是事实。这对我们的投资有哪些启发呢？那就是要想实现长期收益，就要做到每一年都不要亏钱。但是现实是什么样子的呢？拿咱们币圈为例，散户在大部分年份都是亏钱的，只有碰到4年一次的牛市才有可能赚钱，而且也只是有可能，这样长期来看，散户亏钱的概率就变的极高。为什么会这样呢？因为散户们只对挣大钱感兴趣，如果每次都挣钱但挣的不多，大家就是觉得很没有意思，正是因为如此，散户们才会冒着极高的风险博超额收益，其实这样做就是和概率作对。和概率作对，一次两次还可能侥幸赢，但时间稍微一长，输是个必然的结局，因此，我们每一次交易的首要任务不要要赚多少，而是要考虑怎么才能不亏钱。现在又有同学说了，我就喜欢超额收益的刺激，没有超额收益我连觉都睡不香，这该怎么办呢？很简单，那就是你要做好亏损100％的准备，仅此而已。

11月13日比特币行情：16000已破，17000你好！

发表于 2020-11-13 分类于 行情分析

过去24小时，整个市场的交易量1156亿美金，比前一天略多，现在是北京时间11月13日的上午08点31分，BTC即时价格16336USDT，下面是具体分析。

1）经过一周左右的盘整，日线再次开启上涨趋势，目前已经突破了16000-16300压力区间，下一目标将是17000-17150区间，日内如果要回踩，可能的回踩位置16150-16300区间，当然，也可能不回踩，但目前不管是4小时还是1小时都已经有了超买迹象，因此是有回踩需求的。另外，日线并没有开始加速，见顶迹象未出现，那我们就看这波能到哪里吧。

2）BTC不断拉升，一片牛市景象，与BTC相对应的是其他币种，汇率不断走低，这时候就有同学想着抄底其他币种，以求日后的超额收益，那么今天咱们就来谈谈抄底哲学。说起抄底，华尔街有一句经典谚语，叫做Don’t catch a falling knife，中文意思是不要接飞刀，意思是说谨慎买入下跌中的股票，因为下跌中的股票可能会跌到让你怀疑人生的地步。举个例子，熊市中很多币都会跌90%以上，甚至有些币会跌99％、98％等，如果一个币跌了98％，就是下跌了50%之后，又跌掉了96%。也就是说，如果在下跌了50％的时候你认为抄底的机会到了，那么后面还会下跌96％，其实这等于是归零了，你想一想这是多么可怕的事！另外，从价值投资上来说，币圈的大部分币都不具备价值投资的条件，接飞刀的时候你一定要想清楚这样做可能的后果是什么。那么这是不是说币圈就不能抄底呢？不是的。虽然我们不推荐抄某一个币的底，但可以抄市场的底。什么是抄市场的底呢？那就是等市场大跌时买入那些比较热的币，比如ETH等。比如312就是很好的抄底机会。当然，312是几十年不遇的全球金融事件，以后再发生的概率极低，我们拿它为例只是为了更好的说明问题。市场大跌市，大家的情绪一般都会非常的悲观，币价也会偏离正常价格很多，你这时候抄底，就很容易成功。当然，这时候抄底也是需要一定的胆量及技巧的。总之，抄底听起来很美好，做起来却很难，所以我一般是不建议大家抄底的，宁愿成本高一些，也不要危险多一分，这才是我们散户应该坚持的基本原则。

2）咱们之前说过风险就是不确定性，但如果想要给风险下一个准确的定义，似乎又很难，因为不同的人对风险的认识不同，有人认为风险是亏损的可能性，有人认为风险是波动大小，有人认为风险是黑天鹅发生的概率……正是因为如此，风险才不能被量化，这样会导致什么后果呢？那就是你能觉察到风险的存在，但是感觉没事，应该不会发生在自己身上。也就是说，你了解风险但并没有真正理解风险，并且认为风险离自己很远，也正是由于风险的这种看不见、摸不着、难以感知的特性，让很多用户甚至是专业投资者吃过亏。如何解决这一难题呢？那就是我们在投资时不要去考虑风险本身，而是直接去衡量风险的结果。比如你要买入某个币，不用花费太多的时间去想这笔交易会不会亏钱，而应该问问自己这笔交易亏损50％时我有应对的办法吗？是止损还是加仓以摊薄成本？值得注意的是，无论你选择那种方法，只要你事先考虑清楚了，那就是好方法，这也是应对风险所推荐的方法。当然，不让你考虑风险本身并不是让你闭着眼睛买买买，而是要你明白考虑风险的结果更重要而已，当你知道了结果比本身更重要时，能更好的做好风险管理。

11月8日比特币行情：合理且必要的回调

发表于 2020-11-08 分类于 行情分析

过去24小时，整个市场的交易量1434亿美金，比前一天略多，现在是北京时间11月8日的上午09点05分，BTC即时价格14873ISDT，下面是具体分析。

1）昨天分析说只要不跌破14000点，后市就有可能继续创新高，有同学问我是不是写错了，因为当时的BTC还在15650附近震荡，距离14000有1600多刀，你看，这不一晚上就到14000附近了嘛！那么现在行情如何了？既然没有跌破14000点的支撑位，那咱们就把这次下跌看成正常的回踩就行，且由于前段时间一直在拉升，导致大量散户做多（当时在圈子里有提醒大家），这次回踩也显得非常有必要。另外，周末灰度没有继续买入BTC也是下跌的一个原因。那么下跌结束了吗？现在看是止跌了，但后市仍有可能测试14000-14400区间的支撑，若测试不破，则是买入的好机会。上方压力分别在14940和15130，每突破一个支撑位，后市收复跌幅的概率就会变大一些。

2）对弈大多数散户来说，“拿着不卖”看似是一种信仰，但也是严重亏损的原因，我们今天就来分析一下这种思路的好处和坏处。“拿着不卖”多发生在熊市或者一段下跌趋势中，由于人的损失厌恶的特性，大家往往会在被套了之后选择坚持拿着，在坚持的同时，还经常用某一个成功的例子来为自己打气。而在牛市中，大家都有落袋为安的偏好，这时候更倾向于有了利润后就卖出，到了牛市后期，由于长期被牛市气息熏陶，又恰好遇到下跌被套，你又不想卖了，期待着后市还有新高，就这样一直拿到熊市……现在我们可以讨论“拿着不卖”的好与坏了。其实结论已经很明了了，那就是在熊市末期、牛市初期和牛市中期是有好处的，在牛市后期、熊市初期和熊市中期没有好处。你看，“拿着不卖”本就是一种投资方法，本身并没有好与坏，关键是你要用在合适的时间里。你可能会说，这不是废话吗？我要是知道周期，我当然也知道何时该拿着何时该卖出。不错，了解周期当然是最重要的技能，但“拿着不卖”不仅仅是知道周期就行了这么简单，它背后还有更多心理因素，有的人即使知道熊市已来，由于已经亏损很多，他还是会不想买，还心存侥幸，认为市场还会给自己一次机会。你想一想，有没有这种纠结的时候呢？
