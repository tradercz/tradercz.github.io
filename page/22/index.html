8月7日比特币行情：要提防这种情况发生

发表于 2020-08-07 分类于 行情分析

过去24小时，整个市场的交易量883亿美金，比前一天略少，现在是北京时间8月7日的上午09点38分，BTC即时价格11827USDT，下面是具体分析。

1）BTC最高11900点，但行情没有发生本质的变化，继续在高位震荡，大的震荡区间10500-12000，其他并无太多要说，唯一需要注意的是，BTC在12000附近可能会假突破，引诱突破追的人进来后再跌回去继续震荡。此阶段建议的策略就是拿好已有的现货仓位、降低期货的杠杆并尽量减少操作次数。总结一下，依旧是震荡行情，震荡多久不清楚，但震荡后更看多一点。还有，山寨币和新兴主流币的机会仍在。

2）行为经济学中有一个概念叫锚定效应，它和我们之前讲过的有限注意力偏差、代表性偏差一样，都是会导致人们预期偏差的因素。那么什么是锚定效应呢？我给你举个例子，你去商场买衣服，卖家通常都是给你一个原价和一个现价，你看到现价比原价低了好多，就更可能买下这件衣服，这就是锚定效应。 现在咱们可以给锚定效应一个更加标准的定义了，那就是我们的观点特别容易锚定在过去帮助我们形成观点的那个信息上，哪怕新的信息来了，我们也不太愿意改进，这种现象就是锚定效应。这对我们投资有什么指导意义呢？一、现价和历史最高价差距大的币在未来牛市的涨幅就一定大？如果你这样认为，那你就是在受锚定效应的影响，这个历史最高价就是锚定点。真实情况是价差对后市的涨幅可能会有一点点参考价值，但绝对没有你所认为的帮助大，因此，如果你的某个币正在被深套，清醒一点，不要被早晚会涨回来的幻想所误导。二、如果你在一个币上赚过钱，那么在未来你就更可能继续买入这个币，哪怕这个币的现状已经很糟糕，而你却不愿意接受它的变坏。三、如果你在一个币上亏过钱，那么在未来你就会更加远离它，哪怕它现在已经变得足够的优秀。锚定效应与路径依赖有点像，但锚定效应更加隐蔽，哪怕是一个无关紧要的信息都可能成为锚定点，而你却没有发现，你总是受着锚定点的影响而无法做到随机应变，从而无法客观准确地判断出现状。细想一下，你有过这种经历吗？

8月5日比特币行情：震荡

发表于 2020-08-05 分类于 行情分析

过去24小时，整个市场的交易量885亿美金，比前一天略多，现在是北京时间8月5日的上午09点53分，BTC即时价格11166USDT，下面是具体分析。

1）昨天向上突破11500失败，随后回到震荡区间，那么行情就没什么可说的了，就是震荡行情，震荡区间10500-12100，小区间11000-11300，至于要震荡多久，不清楚，大家保持耐心就好。ETH方面，也属于拉升后的震荡，由于前期比BTC拉升的多，在8月2号冲高回落时出逃的获利盘也多，后市也有震荡回踩的需求。总之，大趋势看震荡，震荡后向上的概率较大，在BTC和主流币震荡调整的时候，机会往往会出现在山寨币以及那几个新兴的主流币上。

2）下面讨论两个问题。一、投资的本质是什么？投资的本质就是以时间换空间实现资产增值的这么一个过程。二、交易的本质是什么？交易的本质就是不断试错、用小的亏损博取大的收益的过程。A在5年前买入了BTC，一直拿到现在，资产翻了数倍，这是投资。B每天都在不断地交易，资产也翻了数倍，B就是交易。现实中除了A和B，还有更多的a和b，其中a在17年高点买入了BTC，到现在还被套着，b每天都在交易，但总是在亏损。很显然，a和b恰好处于两个极端，对于a而言，他信奉价值投资，在最高点入后，被套了3年，该退出的时候不退出，缺少的是对周期的把握，缺少的是大局观。对于b而言，他坚信交易能够使资产增值，但总是在亏损，很显然他不适合频繁交易，他太在意几分钟内的涨跌，他缺少的是价值投资思维。投资和交易都是能够使资产增值的方式，没有好坏高低之分，两者是相辅相成的关系，正如这个世界并不是非黑即白一样，交易与投资也不是完全对立的，交易型选手要有投资思维，投资型选手也要有交易思维，总之，适合自己的才是最好的。你看，要想在这个市场赚钱，一定要先搞清楚自己属于哪种类型，那么你是投资型选手还是交易型选手呢？

1）与前一天相比，行情并无变化，BTC继续在10500-11400区间强势整理，下方支撑10500和10000，值得注意的是，11000-12000区间几乎没有压力，只是在11400附近有一些少量筹码的压力，突破后很快就会到12000附近。现在有两种情况，一种是突破11400后快速拉升到12000，一种是突破11400后继续小幅缓慢拉升不断逼近12000点。很多时候，我们是可以从主力拉升的动作上看出其意图的，如果快速拉升，那就要考虑到主力是否急着出货这种可能（注意是可能）了，策略上我们就要做好减仓的准备，如果是缓慢拉升，一步一个脚印，那么说明市场对后市的看涨预期一致，后面就可能涨的更高，我们可以继续拿着筹码。当然，还有一种情况，就是突发状况导致美股再次崩盘，带动全球所有的金融市场大崩盘，就像312那次一样，那么BTC可能又会回落到10000点之下，只是现在看起来这种概率并不大，我们对后市还是保持看多观点的。这里强调一下，有同学看到我们看多，就去满仓梭哈，甚至还加杠杆，这都是不对的，看多看空只是我们对后市的态度，不代表操作上就一定这么做，毕竟操作是需要考虑风控的（其实真正的原因是我也不知道会回调到哪里）。

2）有相当多的一部分人，只喜欢大涨大跌的行情，只要行情稍稍停下来就会很不耐烦，这有什么问题吗？我认为是有的，他们错误的认为行情总是在上涨或下跌，却不知横盘才是常态，也不理解时间换空间的意义，从而导致他们不具备长线思维，以至于总是在频繁交易。当然，我不是说短线不好，而是说很多人都没有长线思维。 什么是长线思维呢？你在某个价格买入，不去考虑明天是否会涨，而是看好几个月甚至几年后的收益，这就是长线思维。若你买入后，想着明天就要翻倍，这就是短线思维。长线思维对我们的人生有哪些指导意义呢？对，这里说的是人生，而不仅仅是交易。首先，长线思维能够让你保持足够的耐心、踏踏实实的做好今天的事，比如学习，很多知识并不是说你今天学了明天就能用到，有的甚至一辈子都用不到，但你能说没用吗？不能。虽然你用不到，但不代表别人也用不到，知识是没有保质期的，它可以传承下来，直到传给有人的人，即使你学得后没有传下去，至少也是可以丰富你内心世界的。其次，长线思维有足够的容错能力，你做一件事，一开始不知道是对还是错，没关系，将时间拉长，在以后的实践中慢慢验证它。再次，长线思维可以让你抵制当前的诱惑，养成延时满足的习惯，这样才更容易实现长期目标。回到交易中，长线思维可以让你潜下心来去学习那些必备的技能，可以让你抵挡住短期波动的诱惑，可以让你少犯错误，可以使你的收益最大化，一个币前期涨了，现在处于盘整期，但上升趋势没破，那就耐心等待它的二次爆发，不要总是怨天怨地。总之，我们每个人都是需要一些长线思维的，就连那些频繁做短线的，也一定是有一部分仓位长期不动的，而且，当你有了长线思维后再去做短线，遇到涨跌的问题时，或许就有另一种领悟了。

7月30日比特币行情：强势整理，耐心等待

发表于 2020-07-30 分类于 行情分析

过去24小时，整个市场的交易量881亿美金，比前一天减少了约100亿，现在是北京时间7月30日的上午09点18分，BTC即时价格11086USDT，下面是具体分析。

1）行情与昨天相比变化并不大，依然处于10509-11400区间震荡，日线连续两天的缩量震荡，属于拉升后的强势整理，下方支撑10500、10000、9600，上方压力11400、12000，突破12000后，将会加速上攻14000，突破14000，就是20000附近的历史高点了，由于上方几乎不存在筹码上的压力区，后面每一次突破后上行的速度都会加快，但到底能走多远，现在还看不出来。总结一下，大趋势向上，继续看盘整后的上攻，至于会不会再次回测10500、10000等支撑，不清楚，也有可能用横盘代替回调。

2）我们做交易一定会用到各种各样的策略，如果一个策略对当前行情没有用，那就换一个策略，这种思维模式我们称之为策略思维模式。策略思维模式，是一种引导自己寻求策略的思维习惯，是一种元认知能力。值得注意的是，会很多种策略和有策略思维模式并不能划等号。举个例子，诸葛亮善于运用很多的策略，但你不能说他有策略思维模式，为啥？因为他几乎算无失策，从没有因为失败而改进过策略。曾国藩就不一样了，从他一开始当官到后来的领兵打仗，总是在失败，但厉害的是他总是能不断改进自己的策略而将麻烦解决掉，我们可以说曾国藩是有策略思维的。这对我们做交易有什么启发呢？当你在某段时间特别不顺，做一单亏一单的时候，你就应该反思了——会不会是策略出现问题了？有没有更好的策略？实在没有好策略，那可以可以暂时停下来不去交易？然而大多数的情况是，当你通过某种策略赚了一大笔钱时，你很容易会将该策略视为圣杯，在往后的操作中，你会不自觉的使用该策略，要知道做交易是没有圣杯的，你这是路径依赖。当你使用某种方法输了一大笔钱时，你不服气，只是觉得自己运气不好，下次一定能赢，你种想要战胜市场的思维同样不可取，要知道市场是无法被战胜的，越是想要战胜市场的，最后输的越惨。因此，在交易中大家一定要养成策略思维模式，不要有执念，经常反思自己，这并不是要求你掌握很多策略，策略思维只是一种思维方法，是不需要任何基础都能掌握的。如果你找不到更好的策略来应对当前的行情，那么对你来说，暂时不交易就是最好的策略。
