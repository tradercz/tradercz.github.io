1）BTC再次发力突破13350点直逼2019年6月高点1400点，这是很多人预期中的盈利点，也就是说，如果后市有机会运行到14000点，就会有很多人在此下车或者做空。当然，如果顺利突破了14000点，下一个目标将会是17000点附近。走势方面，越接近14000我就越有点看空了，原因是拉升的太快，单从技术面上来讲这是不健康的，这种行情随时都有回撤1000刀的可能，但这次拉升的原因又和大机构不断的买买买有很大的关系，而且散户在这波拉升的过程中并未表现出狂热，这就给了后市突破14000点的潜力。总结一下，越接近14000点越忐忑了，保持敬畏同时也保持希望吧。

2）如果你做短线太久了，就很有可能陷入短视陷阱，那么什么是短视陷阱呢？就是只注意到了触目可及的世界而忽略了更大更广阔的未来（当然，词和解释都是我自己造的）。套用到交易上来说就是许多短线高手往往都会在单边行情中被洗下车，这也是我今天要讲的重点。拿现在的行情来说，14000仿佛是个很强的压力位，而且BTC从4000、5000到6000再到现在的13000多点，涨幅已经非常可观了，如果你只做短线，14000点就会是个很好的获利点，如果后市真的涨到了目标，你就很可能获利下车。那么如果你用周线看呢？行情才刚刚起步而已，即使碰上1000-2000刀的回调，周线上也只是个小阴K线而已，但对于短线交易这可能又是一次熊市。当然，我这么讲并不是说短线交易不好，恰恰相反，我认为短线思维很重要，因为只有学会了短线才能做好长线（牛市末期必须要有短线思维），但在大周期的牛市来了之后，除了做做短线，每个人还都应该拥有长线思维，只有这样你才可能经受住牛市中一次次大回调的考验，也只有这样，收益才会尽可能的最大化。

发表于 2020-10-27 分类于 行情分析

过去24小时，整个市场的交易量1037亿美金，比前一天增加了300亿美金，现在是北京时间10月27日的上午08点48分，BTC即时价格13081USDT，下面是具体分析。

1）昨天BTC震荡了一天，没有进展，仍处于12500-13350这一稍大点的区间，凌晨最低跌至12765，测试了12720这一低点，后市可能会在12765-13350区间继续震荡，在这种行情下，我们要做的就是挂好单等待。如果你要问震荡后的走势，我更倾向于向上突破，但其实是不好判断的，即使现在下跌1000多刀，也还是上升趋势，而如果后市在12500-13350区间没有踩实，或者是美国大选出现了意外，都是有机会向下测试的，现在只能说上升趋势没有改变而已。因此，在不可控因素增多的情况下，我们尽量不要去猜震荡后的走势了，做好风控，耐心等待。

2）我知道有部分同学每天都在交易，也就是在做日内级别的短线，那么做日内单需要注意哪些问题呢？我认为主要有以下几点：一、日内交易最重要的不是多盈利，而是少亏损，也就是风险第一，盈利第二。既然你做的是日内单，就不要想着一单能赚上30-50％，你每单只要在弥补交易手续费后还能盈利1个点以上就可以了，那些每天都在朋友圈和群里晒获利几十个点甚至翻了几倍的单子的不是交易，而是赌博（而且这里的水很深的，有专门的APP可以随时生成获利单给你看，他们的目的就是为了让你心动）。二、如果你肠胃不好（回想一下咱们之前讲过的肠脑的功能）或者心情不好，一定不要开始交易。三、尽量只做一种标的，比如只做BTC，这样会因你更加专注而增加盈利的概率。四、每天都手动记录标的的各种数据，比如记录BTC的价格、波动率、永续合约费率，多空比，持仓量等等，注意是手动记录而不是在APP里看，因为手动记录和只用眼睛看是完全不一样的，你记录的时间久了自然就有感觉了。五，使用虚拟盘大量的练习如何止损（止损的动作一定要经过大量练习后才真正的属于你），在这一过程中，你可以乱开仓，但一定要学会果断止损。六、刻意的开仓，比如故意重仓练习心态，尽量在多种行情中做交易。七、经过了以上几步后，你现在可以做减法了，只用你习惯的方法做交易，越纯粹越好，只专注于当下。如此这般，你的短线交易水平一定会上一个新的台阶。

10月26日比特币行情：看多的理由依然存在

发表于 2020-10-26 分类于 行情分析

过去24小时，整个市场的交易量725亿美金，几乎与前一天持平，现在是北京时间10月26日的上午08点47分，BTC即时价格13030USDT，下面是具体分析。

1）昨天突破13200压力位后遇到一小部分获利盘抛售，日线上形成了类似十字星的一根K线，这是行情见顶的形态之一，但我们要知道，价格必须和量能配合分析才能说明市场的真实情况，我们看昨天的量能，是缩量的，这就说明行情还没有见顶迹象。结构上，处于12500-13350区间，下方第一支撑位12800，只要还在12800上方运行，震荡后依然存在较大的概率继续上行，即使跌破了12800，也仅仅是日线级别的回调，下方12000-12200区间仍旧有较大的支撑。总结一下，看多的理由依然存在，期待震荡后的表现吧。

2）你有没有这种经历：一旦交易就停不下来，一旦止损就会很难受……你有没有想过为什么会这样？想要找出原因，就得先弄清楚一个心理学上的概念——随机激励。所谓随机激励，就是结果未知的激励方式。比如抓娃娃机，你总是觉得下一次就能抓到，比如某音小视频，你总会认为下一个会更精彩，类似的还有盲盒、抽奖等，他们都是运用随机激励的好手，你之所以在它们面前很难停下来，是因为每一次奖励时你的身体都会分泌多巴胺，多巴胺是会上瘾的，这就是随机激励具有成瘾性的原因。同理，交易时停不下来也是期待随机激励的结果，这也不难理解止损为什么这么难了，你每一次止损就相当于人为终止了即将到来的随机激励，你会认为止损是反人性的。如何解决这一难题呢？我从心理学的角度给你提出三个解决方案。一、调节环境，比如建立互助小组相互监督，只做正确的事。不要觉得这没有用，环境对人的影响是非常大的，你认为你是自由的，你以为你有自由意志，其实都是你所处的环境让你这样认为的。如果你想做到坚决的止损，有一个好的环境能起到事半功倍的效果。二、每一次成功止损或者每做一次成功的交易就为自己欢呼（或播放某一个你特别喜欢的音乐、音效等），特别是成功止损时，不断强化止损是正确的这一选项，久而久之，再遇到该止损时就就很能很容易做到止损。三、利用虚拟盘做大量的练习，不要考虑盈利多少，只在乎成功止损了没有，当你每次都能做到成功止损时，你会发现盈利很容易就能实现。值得注意的是，上面说的是有前提的，那就是你已经有了一套经过市场验证过的盈利方法，也就是说，本文的对象是已经有了盈利方法但无法做到停止交易或止损的交易者，如果你还没有达到这一水平，不要急，先去建立自己的交易体系吧。

10月20日比特币行情：站上11750点后，下一目标就是12000

发表于 2020-10-20 分类于 行情分析

过去24小时，整个市场的交易量723亿美金，比前一天增加了150多亿，现在是北京时间10月20日的上午08点22分，BTC即时价格11720USDT，下面是具体分析。

1）昨天的日线很漂亮，放量阳线（量能其实不多，只是与前两天相比是放量），现在BTC已经进入了11720-12050的震荡区间，能不能站稳11720还有待观察，至少现在是情况很不错，接下来就是12000-12050的目标区间了。行情到了这里，多空分歧已经出现分化了，多方认为能站上12000直至前高12468，空方认为行情已经差不多到头了，看多看空的理由都是有的，我也没法给出明确的观点，主要是多方上攻的意图并不明显，显得无力，空方又毫无防守的意思，如果非要我说出我的看法，还是看多更多一些，但对行情最终能走到哪里不清楚，那就耐心等待吧。还有，多空都没有大动作也可能和即将到来的美国大选有关，不仅BTC，黄金外汇股票等各个市场也都在等美国大选尘埃落定。对于散户而言，这个时候就不要赌了，给市场一点时间，耐心等待。

2）先来看一个开脑洞的问题。有一天，你来到深圳的海边，把一杯水倒进大海里。过了5年，你去美国旧金山的海边，用杯子舀起一杯海水。请问：5年前你倒入深圳的那杯水，有多少会出现于你在旧金山的这个杯子里？你是不是觉得很扯淡？距离那么远，还相隔了5年，答案肯定是零啊！但是正确的答案应该是是1000个水分子。这1000个水分子是怎么怎么来的呢？是通过概率的计算得来的。另外，有同学害怕做飞机，因为我们总会听到飞机失事的消息，那么飞机失事的概率究竟有多大呢？如果你每周坐一次飞机，那么你连续坐五万年，才会遭遇一次飞机失事的惨剧。好了，我们通过上述两个例子，让你明白了概率思维的用法，下面我们接着讲飞机失事的例子。既然飞机失事的概率很低，那是不是说坐飞机就非常安全呢？从概率的角度看是这样的，但对于某个人来说可不一定，哪怕飞机失事的概率很低，一旦发生对于当事人来说就是100％的事，这时候我们又引出了随机性这一概念。另外，第一个例子中之所以会有1000个水分子，也是随机性在起作用（水分子的随机运动）。明白了概率思维和随机性后，我们可以从中得出什么启发呢？一、交易时只做大概率的事情有助于交易的成功。二、大概率不代表一定会成功。三、如果你做了小概率的交易还成功了，请记住这是随机性在起作用，你一定要懂得见好就收（请复习一下我们之前讲过的随即奖励的概念，我告诉大家不要对随即奖励上瘾）。四、不要看到别人偶尔发了财就想着要复制他的方法，同样的，如果你侥幸发财了，也不要轻易地指导别人。总之，拥有概率思维并理解随机性无处不在的事实更有利于你成为一个成熟的交易者。
