行为金融学从微观个体行为以及产生这种行为的心理等动因来解释、研究和预测金融市场的发展。这一研究视角通过分析金融市场主体在市场行为中的偏差和反常，来寻求不同市场主体在不同环境下的经营理念及决策行为特征，力求建立一种能正确反映市场主体实际决策行为和市场运行状况的描述性模型。

行为金融学虽然试图深挖金融市场运作背后的奥秘，但并不系统也不透彻，因此现今成型的行为金融学模型还不多，研究的重点还停留在对市场异常和认知偏差的定性描述和历史观察上，以及鉴别可能对金融市场行为有系统影响的行为决策属性。究其原因，也许就像指数基金教父约翰·博格尔所说的那样：“在这个行业里摸爬滚打了55年，我依然对如何预测投资者的心理一无所知。”

值得指出的是，人类对于股市波动规律的认知，是一个极具挑战性的世界级难题。迄今为止，尚没有任何一种理论和方法能够令人信服并且经得起时间检验——2000年，行为金融学理论创始人之一、耶鲁大学经济学教授罗伯特·席勒在《非理性繁荣》一书中指出：“我们应当牢记，股市定价并未形成一门完美的科学”；2013年，瑞典皇家科学院在授予罗伯特·席勒等人该年度诺贝尔经济学奖时指出：几乎没什么方法能准确预测未来几天或几周股市债市的走向，但也许可以通过研究对三年以上的价格进行预测。

当前，有关金融资产定价和股票市场波动逻辑的代表性理论，主要有如下几种：凯恩斯选美论、随机漫步理论、现代资产组合理论(MPT)、有效市场假说（EMH）、行为金融学（BF）等。

1月23日比特币行情：短期反弹

发表于 2021-01-23 分类于 行情分析

过去24小时，整个市场的交易量1589亿美金，几乎与前一天持平，现在是北京时间1月23日的上午08点00分，BTC即时价格32882USDT，下面是具体分析。

1）昨天BTC最低跌至28850，目前已经反弹至32500上方，日线看回调至30日线和69日线之间，回调还是比较充分的，就像咱们之前说的，这次下跌可能是牛市中最好的一次建仓机会。结构上，下方支撑32500，上方压力34000。后市走势方面，如果能站稳32500，那么还有可能向上测试33000-36000区间，若站不稳，下来测试前低的概率变大。总结一下，下跌后的反弹，目前已经接近目标位，日内需关注32500的得失。

2）这是『行为金融学』系列文章的第一篇，我们重点来谈谈传统金融学与行为金融学的区别。首先先要搞清楚什么是金融，按教科书中的定义，金融就是资金的流动，在市场主体的作用下，资金从一方流到另一方的活动就是金融。这里里面有一点需要考虑，那就是这些资金是通过什么工具流通的呢？很多，我随便举几个例子——存款凭证、股票、债券等等都是金融工具。现在，你大致了解了金融的定义，那么咱们就来看看传统金融学与行为金融学的区别有哪些。很简单，传统金融学研究的是，市场“应该是”什么样，是长期的趋势和规律，而行为金融学研究的是市场“实际是”什么样。举个例子，传统金融学家与行为金融学家正在走路，突然听到有人喊，快看，地上有一百块钱。传统金融学家就会说，如果地上有钱，早就被人捡走了，所以地上不应该有钱，而行为金融学家会说，怎么不可能啊，他跑去一看，果然有一百块。再比如，对于BTC走势，传统金融学家会说，长期看BTC一定会涨，而行为金融学家会说，今天BTC可能会跌。现在，你大概明白了这两门学科的不同了吧，也就是说，虽然都叫金融学，但这两者是有本质的区别的，传统金融学认为市场是有效的，是无法预测的，而行为金融学认为市场在某些时候是无效的，是可以被战胜和预测的。当然，这并不是说传统金融学就落伍了，目前这两门学科都在持续的发展中，并对我们的经济活动产生着巨大的影响。现在，我们来点实际的，行为金融学究竟有什么用？简单的说，行为金融学最擅长的是投资实战，很多交易策略比如“量化交易”都是在它的理论基础上制定出来的。好了，这就是今天的内容，看过后你应该对行为金融学有一个初步的印象了，但你要明白，这只是皮毛，如果你想更深入了解，一定要去系统的学习才行。

1）比特币再次下跌，凌晨零点跌至33400点后遇到抄底盘开启了反弹，但截止发文时仍然只是反弹，若后市不能站上36000点，再次向30000点下压的可能性就会增大。结构上，上方压力36000点，下方支撑34000点。总结一下，震荡行情，短期反弹，后市想要走强，必须尽快站上36000点。

2）最近发现很多同学对仓位管理还不理解，即使之前咱们说过了，但今天还是再谈谈这个话题吧。首先是为什么要做仓位管理？原因很简单，那就是无论我们使用什么方式，都无法精准的预测市场未来的走势，我们不可能买在最低点，卖在最高点，我们更不能保证买入后就一定会盈利，也就是说，很多时候预测是不可行的，但预防是很必要的，而仓位管理就是预防的最佳手段。那么仓位管理要遵循哪些原则呢？还是以我为例。一、永远不会满仓（目前8层仓位），永远手握一部分现金。需要注意的是，在币圈梭哈（满仓）的人很多，但科学的仓位管理认为，梭哈不是一个好主意。二、很少空仓，永远不要给自己留下踏空的可能。三、分批买入卖出（非常重要）。三、适当的币种配置，比如其他币的比例不会大于BTC和ETH。四、3-6个月做一次再平衡。最后是针对新手的两个概念，一个是对“层”的解释，就是已投入金额占全部可投入金额的比例，我8层仓就是用了80％的资金买币。另一个再平衡，我们以法币为标准，比如BTC和ETH的比例是2：3，由于价格的变动，过段时间BTC：ETH变成了1：1了（BTC涨的比ETH多），那么我会选择卖出一部分BTC买入一部分ETH，将比例再次调整至2：3，如有其他币种，做同样的处理，需要注意的是再平衡策略无需太频繁，我的经验是半年一次即可。以上就是我的仓位管理原则，是不是很简单呢？但是它却很重要，可以说是我能够稳定盈利的最大的武器，希望你也能很好的运用它。

2）经常有同学被套后来问我怎么办、要不要止损，说实话，这个问题非常难回答，今天就以我为例，来谈谈如何解决这一问题。如果我被套了，我会先看看趋势是不是已经发生了变化，如果趋势没变，那就继续持有就好了，当然，很多时候趋势有没有变化是很难判断的，如果真的搞清楚了就不会有要不要止损的困惑了。接着我会考虑当初入场时的条件是不是发生了变化，比如我在一根放量阳线后入场，可后来跌破了这条阳线的低点，那我肯定是要止损离场的。最后，我还会考虑一点，那就是做这单是盈利的概率大不大，如果入场时我认为这单80％都能盈利，可事实证明我亏损了，那么我也会坚决止损，如果当时我认为这单短期内盈利的概率不大，但时间长一点就会大概率盈利，那我一般不会止损，也就是说，我能承受被套一段时间。当然，我们在很多时候是搞不清短期走势的，但如果我能承受这一单归零，我也有可能选择不止损，因为小单子被爆仓可以使我警醒，这样就避免了我在大单上犯错误。 总之，要不要止损这一问题，我们在坚决止损的大原则上，可以有小小的变通。另外，我们之所以不喜欢止损，是害怕止损后行情却回来了给我们造成心里上的踏空，这个一时难以解决，只能继续修心了。

1）依然是震荡行情，小时线与4小时线的几条均线已经缠绕在了一起，今明两天大概率会选择方向。结构上，下方支撑34500-35000区间，上方压力38000-38500区间，跌破或突破都有可能使短期趋势得到确认，跌破34500测试前低30000点，突破38500上攻前高42000点。总结一下，耐心等待方向的选择。

2）在交易中，我们经常遇到恐惧、愤怒、贪婪、报复和上瘾5种情绪，不管你承认与否，我们每个人都会遇到其中的一种甚至多种，可以说他们是交易杀手，带着这些情绪做交易，100％不会成功。那么如何消除这些情绪呢？我们一一来破解。恐惧源自于无知，我们对未来行情的不确定导致了恐惧，每当下跌时，最先投降的一定是没有交易经验以及没有经历过牛市的新人，解决恐怖的方法很简单，就是多听多学多练。80％的意外事件会导致愤怒，我没有想到比特币会跌这么惨，很愤怒，我没有想到比特币会涨这么多，也很愤怒。解决愤怒的法宝是学习一点斯多葛哲学，永远考虑到最极端的事情会发生。贪婪的本质是不切实际，你看到某某赚了5倍、10倍，你觉得你也一定能赚那么多，但事实是即使你使用了相同的方法，也很难达到同样的效果，因为能够赚钱的那种行情已经不在了。为了应对贪婪，你应该从幻想回到现实，翻倍大神的操作可以借鉴，但一定不要将其设为目标，特别是最近的行情，每天都有很多翻倍币诞生，如果你不切实际的想要赌对，就很可能由于频繁换车而踏空所有的币。报复心理是最难克服的，也是交易最大的杀手，一段上升趋势中，我踏空了，就很容易去做空，这就是报复心理。为了解决报复心理，我们需要多看哲学、心理学一类的书籍，学会感恩、学会接受、学会臣服。关于上瘾，咱们之前说过，是因为随机激励的存在，这次不盈利，下次一定会盈利，这种赌博心理会促使我们频繁的交易下去，永远不会停下来休息，不管行情给不给机会，每天都在交易。至于如何克服随机激励带来的频繁交易这一问题，我推荐你看我之前写过的文章， 这里就不重复了。此外我还推荐你阅读我们的《行为金融学》系列文章，它对我们的交易非常重要。

1月16日比特币行情：震荡行情

发表于 2021-01-16 分类于 行情分析

过去24小时，整个市场的交易量1495亿美金，比前一天增加了100多亿，现在是北京时间1月16日的上午08点28分，BTC即时价格37237USDT，下面是具体分析。

1）比特币突破38000后又跌了回来，并没有冲击前高，仍然在30000-42000区间震荡，后市也可能继续延续这种震荡行情。需要注意的是，4小时线的MA30、MA60以及MA90三条均线已经从密集逐渐呈现多头发散状态，只要比特币能继续在35000上方震荡，那么后市继续走高几乎是必然的。总结一下，继续看好震荡后的行情。

2）你是否有这种体会，那就是总觉得自己手里的币不涨，没有买的币却在疯涨，也就是说，你总是觉得自己买什么都不对。我今天要告诉你的是，这是一种错误的心态，你得尽快改正，否则在牛市的下半程你很可能会因为频繁跳车而错过很多的机会。你之所以频繁换车，就是控制幻觉在捣乱，你总是想吃掉所有币的涨幅，你总是觉得自己能够吃掉所有币的涨幅，这种想法是不现实的，要知道到了山寨币的季节，阿猫阿狗都会起飞，你不可能抓住所有币的涨幅，你只要抓住1、2个币的涨幅，就足以完成阶级的跃迁，而频繁跳车的收益并没有你想象的那么丰厚。我的建议是，买入几个看好的币就不要动了，并且我也是这样做的。我从去年的12月1号就开始给会员推荐其他币种，如果你是那时候买入的，这中间你即使什么都不做就可以有不错的收益，很多已经翻倍了，这种慢就是快的投资策略在牛市的下半程效果会更加的明显。你之所以追涨杀跌，一是你没有经历过牛市的疯狂，二是你的控制幻觉在作怪，很多时候，只要你将心态放正，投资就不需要太多的技巧的，也就是说，投资很简单，但反人性。

1月15日比特币行情：多头趋势延续

发表于 2021-01-15 分类于 行情分析

过去24小时，整个市场的交易量1328亿美金，几乎与前一天持平，现在是北京时间1月15日的上午08点22分，BTC即时价格39418USDT，下面是具体分析。

1）比特币再一次接近了前高，那么接下来冲击前高就是水到渠成的事，至于能否成功，咱们拭目以待。结构上，下方支撑38000，上方压力40700附近（筹码密集区），突破了40700，冲击前高的概率就会增加，大家可以关注该点位的得失。链上数据方面，大户趁着这次大跌在加仓，部分矿工和散户在抛售，这对长期走势是个利好。总结一下，在成功突破38000后，本次下跌已经结束，接下来就看何时能创新高了。

2）咱们经常强调不要凭感觉开单，不要冲动开单，不要盲目开单，没有机会的时候就休息，今天就以我这几天的操作为例来对这个话题做个补充。回到1月11日，BTC大跌，我多单止损后就暂时休息了，到了13日晚间开始，BTC开始有走强的迹象，这中间我都没操作，直到14日凌晨，大量的稳定币流入交易所，这是一个很重要的信号，我将这一消息发在了会员群，让大家关注36000点的突破情况。随后我自己在比特币突破36000点后买入做多，正式开始了下跌以来的第一笔短线操作。你可能会说，你这算什么操作啊，我看到某大神在30000点抄底，人家才是真牛。是的，我的操作一点也不牛，我也不会去猜底，所以我不会在下跌的过程中去做多，除非有二次测试的机会，但这就是我的操作风格，很适合我。我将它拿出来，是为了告诉你我没有凭感觉开单、没有冲动开单也没有盲目开单，同时我认为我们每个人都要有自己的开单规则，这个规则最好不能是某个人的一句话或者某一条消息，就像上面那条稳定币消息一样，它之所以成为了我的信号，是我综合了当时的盘面情况，如果你对当时的盘面不了解，就不要轻易将它当作你的开单信号。总之，我们要学会将杂乱的信息融合到自己的操作系统中，从而形成自己独有的交易框架。
