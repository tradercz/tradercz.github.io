1）昨天量能继续扩大，BTC也强势突破了日线级压制，目前已经重新站上10000点，直指10500强压力位。关于后市，还是继续看多，但在突破10500之前，BTC可能有一波回踩，理由如下：一、10500强压力，有大量在空头在此防守。二、目前BM上BTC永续合约资金费率0.037，说明做多的比较多（正常费率0.01左右），不利于拉升。三、今天CME上出现了将近300（9645-9925）美金的缺口，大多数情况下缺口都会回补，如果回补，BTC现货会回到9615附近。当然，强势行情下，一切皆有可能，我们只是预测其可能的走势，但千万不能有执念，如果不回踩直接突破，或者拉升至10500附近再跌回来，都不要感到太意外。

7月23日比特币行情：短期多头确认, 等待中期多头的开启

发表于 2020-07-23 分类于 行情分析

过去24小时，整个市场的交易量592亿美金，几乎和前一天持平，现在是北京时间7月23日的上午08点34分，BTC即时价格9513USDT，下面是具体分析。

1）先说说ETH，今天早间ETH强势突破253点的压制，一举拿下了8个多点的涨幅，位居主流币之首，多头趋势很明显，下一目标285-287附近。在ETH的带动下，BTC也成功突破了9470点的压制并站上了9500，单看BTC，多头动能并不是很足，但在ETH为首的主流币的带领下，可以说短期多头趋势已经确认了，BTC接下来的目标位9700附近，其中9700是日线前高已经筹码交投高峰，9760是4小时下跌趋势的0.618，如果9700-9760能顺利突破，则中期多头趋势将会被开启。向下看，昨天在分析中说了，两个月的震荡使得9180-9220区间有可能成为未来一个非常重要的支撑区域，另外9300、9400也都是小支撑。总结一下，短期多头趋势基本确认了，建议回调做多，回调目标9420附近。

2）A和B抛硬币，每人抛5次，A的结果是正、反、反、正、反，B的结果是正、正、正、正、正，现在我问你，A和B哪个更符合实际情况？我猜大部分人都选A吧，因为B太工整了。我现在将问题改变一下形式，A和B发生的概率哪个更大？只要你有中学的数学知识，就能很快算出来，在硬币没有被人为做手脚的情况下，A和B发生的概率一样大，都是2的5次方分之一。发生了什么？我问你哪个更符合实际情况时，你很容易选择A，就说明你有代表性偏差。与有限注意力偏差一样，代表性偏差也是行为经济学（行为金融学）中一个重要概念，它说的是我们总是默认某些事有特别的特征，但事实并不是这样。那么代表性偏差对我们的投资交易有什么样的指导意义呢？一、如果一个币过去表现的很好，那么将来它也一定表现的很好，这种思维是不对的。二、如果某个交易员过去连续数周的胜率很高，那么他将来的胜率也会很高，这种思维也是不对的。三、每个人都有代表性偏差，专业人员也不例外，因此传统金融市场就有一个这样的策略——卖空那些分析师长期看多的股票，买多那些分析师长期看空的股票，结果也是可以盈利的。至于该策略适不适合币市，你认为呢？四、这个市场中没有常胜将军，也没有圣杯，更没有捷径，不要想着跟着谁就能发财，只有不断提升自身的素质与认知水平，不断完善自己的交易系统，才有可能得到市场的回馈。

7月22日比特币行情：多头再起？

发表于 2020-07-22 分类于 行情分析

过去24小时，整个市场的交易量613亿美金，比前一天增加了约100亿，现在是北京时间7月22日的上午08点33分，BTC即时价格9363USDT，下面是具体分析。

1）昨天下午3点后，BTC开始发力，虽然仅拉升了200美金，但也成功站上了60日线，给横盘2个多月的市场带来了希望。现在行情如何了？首先，近2个月的下降趋势线被成功突破，多头势能增强。其次，两个月的震荡使得9180-9220区间有可能成为未来一个非常重要的支撑区域。再次，多头目标上移，目标区间9470-9520。以上三点就是昨天拉升的结果，那是不是说多头趋势已经确立了呢？并没有，现在大结构仍然在8600-10500，小结构9300-9470，后市只有突破了9470-9500压力区间，我们才可以说短期多头趋势的确立。日内需要关注的是这波拉升后的回调情况，小回调可以，不回调也可以，但如果跌破了9180，则说明这波上攻无效。总结一下，大结构未变，小结构有走强的契机，后市需关注9300、9180等位置的支撑情况。

2）在概率论中有一个非常重要的定理——大数定理，它说的是当样本数量趋于无穷时，样本的均值一定是趋向于样本的期望值的。拿交易来说吧，如果你的胜率是60％，当你交易的次数越多（理论上是无数次），你的实际平均胜率越接近60％。那么大数定律对我们的交易有什么样的指导意义呢？一、我们大部分人的胜率只有50％，优秀交易员的胜率可以达到60％、甚至70％，要想稳定盈利，你需要将胜率提升至50％以上，但也不要盲目追求80％甚至90%的胜率，不现实。二、大数定律可以让我们用整体的确定性来对抗局部的随机性，一次两次交易失败不可怕，但一定要严格止损，只有严格的止损才能保证你有尽可能多的机会继续交易下去，最终实现均值回归。三、当数据量很少的时候，一件事发生的频率可能和它的真实概率相差很大，如果你在一段时间内的胜率非常高，千万不要骄傲（请回顾咱们讲过的随即奖励那篇文章），长期看来，你真实的胜率并没有那么高，短期内胜率高只是由于交易次数太少。四、如果你的胜率实在是太低（低于50％），唯一能保证你能盈利的办法是减少交易次数（在不考虑盈亏比的情况下只做中长线）。

2）行为经济学（行为金融学）里有一个概念叫做有限注意力偏差，是预期中偏差的一种，说的是由于注意力是非常费能量、非常稀缺的资源，我们不可能一直关注某一件事，这就可能导致我们在注意力上出现厚此薄彼的情况发生。哪些情况能影响到我们的注意力呢？首先是最近发生的事情会影响我们的判断。以币圈为例，2018年很多人被套，有一个很重要的原因是他们对2017年牛市的印象太深刻了，这就就导致大家对风险判断的偏误。今年312大跌后，很多人被打懵了，迟迟不敢买入，甚至到BTC都翻了3倍了还沉浸在熊市思维中。前段时间，BTC站上了9000直冲10000点，很多人便忘记了风险一把梭。以上这些都是近期事件影响有限注意力的表现。其次，媒体反复报道的事件也会引起大家的有限注意力。比如你会在不知不觉中购买做了广告的产品，你会被大V的盈利单所吸引等。再次，突出事件也会影响大家的有限注意力。比如你非常清晰的记得谁谁谁购买了某个币而实现了财富自由等。你看，正是由于我们的注意力有限，以至于我们在犯错时都没有意识到。现在你了解了有限注意力偏差，那就以后的投资中，是不是要尽量避免一下呢？

7月19日比特币行情：为何做到知行合一非常难？

发表于 2020-07-19 分类于 行情分析

过去24小时，整个市场的交易量449亿美金，比前一天减少了约50亿，现在是北京时间7月19日的上午09点28分，BTC即时价格9175USDT，下面是具体分析。

1）30、60日线已经走平，无论是月线、周线还是日线，都进入了缩量震荡行情，目前只能等待市场选择方面。大的震荡区间8600-10500，下方8600-8800是支撑，如有跌到位的机会，可做多，破位8600止损，上方10500是强压力位，破不了的话，就不要期待牛市的到来。小结构上，9050-9350，这点空间做的话还不够手续费呢，要做，就找机会小仓位做山寨币吧。

2）你有没有思考过这样一个问题：我们总是在强调知行合一，但却很少有人真正的做到知行合一，这是为什么呢？知行合一为什么这样难？先说“知”的问题，在人类的大脑中，杏仁体是控制我们情感的部位，前额叶皮层是控制我们逻辑思维的部位，杏仁体与前额叶皮层最大的区别是，杏仁体反应快，不太准，有时候会犯错，但是不费能量；前额叶皮层的反应相对来讲比较慢，又非常费能量。当你看到一条疯狗，你会立刻逃走，这就是你的杏仁体在起主导作用，你不会停下来计算一下疯狗攻击你的概率有多高、该不该逃走之类的问题。这时候，指导你行动的主要是杏仁体，这是基因留下来的东西，是能够保命的。但是呢？来到了现代社会，特别是遇到了投资这门新学科，使用前额叶皮层往往比用杏仁体更科学，但由于历史遗留问题的原因，我们往往还会继续使用杏仁体，这就导致了我们更容易在“知”上犯错误。现在来到了“行”这一问题，是什么控制着我们的行动呢？科学证明，是多巴胺。还是上面那种情况，当你遇到一条疯狗，你感觉到害怕，但如果这时候你体内没有分泌多巴胺或者在分泌的过程中出错，那么你也不会逃走。你看到很多“知道该怎么做却没去做”的情况，多是多巴胺没有到位造成的。这对我们的投资有什么指导意义呢？有很多的人对比特币是有信仰的，信仰养成之路一般是从了解开始，在了解的过程中分泌的多巴胺比常人多，就更容易形成信仰，最终将全部积蓄都投了进来。这种做法对吗？现在看大概率是对的，但如果将BTC换成传销或者换成已经归零了的某种数字货币呢？那他可能就不会成功了。你看，多巴胺分泌过的多不能说是对还是错，但终归是有点不妥的，最舒服的是多巴胺分泌适中，但这个“适中”是何其的难啊。现在我们知道了，“知”和“行”是两个完全不同的概念，分别由我们身体的不同部位所控制，想要两者“合一”，实在是难。
