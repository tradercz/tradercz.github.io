12月26日比特币行情：上涨行情再次启动

发表于 2020-12-26 分类于 行情分析

过去24小时，整个市场的交易量1859亿美金，几乎与前一天持平，现在是北京时间12月26日的上午08点01分，BTC即时价格24767USDT，下面是具体分析。

1）昨晚7点，BTC选择了向上突破，经回踩确认，突破有效，至此，BTC开启了一段新的上涨趋势。结构上，下方支撑24200-24570，也就是说，后市可能再次回踩至该区间，但只要不跌破24000，则上升趋势将延续。链上数据方面，昨天拉升前有大笔BTC从Coinbase交易所流出，并且近期多次拉升前都发生过同样的现象，大家以后也可以关注一下这个数据。另一方面，拉升后有BTC从矿工地址流入交易所，但数量不是很大，不必太担心，我们继续保持关注即可，只要买盘力量大于卖盘力量，BTC就会在震荡中上行。总结一下，新的一段上涨趋势开启，但鉴于目前正处于圣诞假期，快速拉升的可能性并不大，更可能是在震荡中缓缓上行。

2）如果说哪本书对我的交易最有帮助，我认为一定是尼古拉斯·塔勒布的那本《反脆弱》，这是一本讲述不确定性以及如何应对不确定性并试图从不确定性中获益的书。先给你说说书中的一个观点，那就是黑天鹅事件比你认为的更容易发生，请记住这句话，它非常重要，甚至能颠覆你的认知。那么如何应对黑天鹅事件呢？传统的方法往往都是加强防范，不过这样的效果并不明显，因为无论你准备的多么充分，黑天鹅还是会发生。在本书中，作者提出了一个应对黑天鹅的新方法——反脆弱。什么是反脆弱呢？它是脆弱的对立面，用一句话概括就是杀不死我就会使我更强大（想一想赛亚人，每一次濒临死亡都会使战斗力大幅提升）。我知道你现在还是不太理解反脆弱，没有关系，你甚至不用理解，重要的是如何在交易中运用它。那么如何运用呢？你要记住一个词——杠铃策略。这个策略的特点是将大部分资金投入到安全性高的标的上，再将少量资金用在高风险标的以博取超额收益。比如我喜欢将大部分资金投入到比特币及少数几个主流币上，再拿出一小部分资金做期货和期权。而有的同学呢？认为比特币太贵了，于是将部分资金都用在了风险更高价格更低的币上，一旦遇到黑天鹅（请记住它比你认为的更容易发生），这些币的跌幅就会比比特币更大，若再没有期货和期权做对冲，就会遭受重大损失。有同学说了，杠铃策略不就是仓位管理吗？不是的，杠铃策略的重点是让我们忽略中庸，重点关注两个极端，而且一旦你学会了杠铃策略，就会发现它太重要了。比如你每天浏览那些大V的行情分析文章与你系统学习金融知识相比，浏览大V的文章就是中庸，你不应该在上面花费过多的时间，你应该通过系统学习提升自己。除此之外，杠铃策略还有很多应用，我就不举例了，由于篇幅的问题，今天只能说这么多，如果你系统了解反脆弱，那么一定要亲自去细读塔勒布的《反脆弱》（它里面的内容比我今天提到的要丰富的多），不过我要提醒你，这本书的语言很散很杂，没有耐心是读不下去的，加油吧。

12月24日比特币行情：震荡行情继续

发表于 2020-12-24 分类于 行情分析

过去24小时，整个市场的交易量2255亿美金，比前一天增加了约300亿，现在是北京时间12月24日的上午08点24分，BTC即时价格22943USDT，下面是具体分析。

1）BTC仍处于区间震荡，震荡区间21800-24300，从最近的走势可以看出，20000点之上的分歧逐渐加大，获利盘在不断的卖出，接手盘又在不停的买入，谁更技高一筹暂只能留给时间。结构上，下方支撑22800，上方压力22600附近，跌破22800就会测试22300附近的支撑。走势方面，预计还会持续震荡。总结一下，20000以来的上升趋势没有被破坏，短期继续看震荡。

2）在昨天的分析中，我们提到了价值投资，尽管你经常听说价值投资，但有可能你理解的价值投资与真正的价值投资还是有差别的，今天咱们就来进一步谈谈这个话题。你需要知道的是，价值投资的祖师爷不是巴菲特，而是本杰明·格雷厄姆（Benjamin Graham，1894 - 1976），几十年前人们谈起格雷厄姆就像现在的人谈巴菲特、查理·芒格一样。查理·芒格有个学生叫李录，他在2020 年出了本书叫《文明、现代化、价值投资与中国》，这本书很好地总结了什么叫价值投资，根据他的总结，价值投资的理念一共有四条，其中格雷厄姆贡献了前三条，巴菲特贡献了第四条，我们一起来看看这四条理念。一、买股票的本质是买公司。投资是为了公司的价值，而不是为了投机。二、忽略市场的短期波动。三、理解所谓“安全边际”，就是当价格远远低于估值的时候再入手。四、打造自己的能力圈，为了合理评估公司的价值，价值投资者必须是个通才，得从宏观到微观、从消费到科技什么都懂才行。现在，你知道了价值投资的理念了，那么我们可以进一步思考，价值投资适用于币圈的普通投资者吗？我觉得很难。第一第四条离普通投资者太远了，因为99％的投资者都是来投机的且他们都不具备甄别好公司、好项目的能力。对普通投资者真正有意义的是二三条，第二条是让你忽略短期的波动，这也是我建议的牛市中的现货策略。第三条说白了就是捡漏，比如你若能在312大跌中抄底，那么今年一定有丰厚的回报。但你要明白的是，这种机遇很可能几年几十年才有一次，你不能对它抱有太大的幻想。另外需要注意的是，价值投资一直有一个陷阱，那就是似乎无论什么标的，只要你一直拿着，都会得到相应的回报，其实这是非常错误的，巴菲特之所以能够盈利，不仅与他的坚持有关，更与他本身能够强大到左右标的公司的运营有关，这对于普通投资者来说是绝对办不到的。因此，对于币圈普通投资者来说，大部分的长期持有都不是价值投资，当然极少数综合素质高的投资者除外。现在，看看你有没有被套还坚持拿着的币，好好想一下自己是在做价值投资吗？

12月22日比特币行情：关注两个重要点位

发表于 2020-12-22 分类于 行情分析

过去24小时，整个市场的交易量1882亿美金，比前一天增加了200多亿，现在是北京时间12月22日的上午08点40分，BTC即时价格22577USDT，下面是具体分析。

1）在欧洲疫情二次爆发的危机下，金融市场全线大跌，BTC也从最高24300跌至21911点，冲高之路暂时被阻。细节方面，昨天16点那根4小时线放量砸下来后，被量能相当的买盘接住，使得BTC从21900反弹至23000点之上，行情暂时止跌，下方支撑22300-22600区间，上方压力23300附近。走势方面，21900有成为本次下跌低点的可能。总结一下，下跌之后的震荡行情，尽管21900可能是本次下跌的低点，但从更大的日线周期看，回调并不彻底，那就先关注短期是跌破21900还是突破23300吧。

2）咱们之前说过，对于交易能否盈利来说，预测涨跌不重要，重要的是有没有一个好的策略，有同学对此不太理解，今天就来解释一下这个问题。以当前行情为例，BTC经过了短时下跌，正处于反弹之中，我们用斐波那契回撤工具测一下便知，BTC已经突破了0.382关键点位，正在测试0.5，那么这时候我们应该如何做单呢？咱们先来预测一下行情，0.5可能被突破，也可能突破不了，这个概率在每个人心中是不同的，我觉得突破的概率是30％，你觉得突破的概率是60％，甚至还有人觉得一定会突破，但我们知道，市场没有什么100％确定的事，一切都是概率而已。如果我们以60％的可能性为例来做策略，应该怎么做呢？你可能会说，60％就是大概率会突破了，我会做多赌突破，但有经验的交易员往往不会这么做，他大概率会在接近0.5的位置做空，若后市突破0.5，空单止损反手开多。你可能明白了，这就是咱们平时讲的看多不做多，而且这里的60％其实是误导你的，即使换成10％也是一样，这不重要。我们接着看，如果后市突破了0.5，空单止损，反手开多，开多的理由很简单，既然突破了0.5，后面就很多能反转了，即使不反转也可能会向上测试0.618，也就是说，0.618是大概率的事情。你有没有发现这里的重点，那就是预测A会不会发生不重要，重要的是如果A发生了（突破0.5），那么大概率会发生B（测试0.618），不错，我们的策略也正是基于此做出来的。当然，你不必先开空再再开多，对于喜欢做多的，也可以等突破0.5做多，对于喜欢做空的，可以在接近0.5的位置做空，突破止损，这就是策略，与预测涨跌的关系没你想象的大。需要注意的是，上面这段话是我在早上5点钟写的，等上午发文时或许就突破0.5了，当然也可能突破不了。

2）很多同学往往会遇到长线拿不住、短线拿太久的问题，就很容易将长线做成短线，将短线做成长线，今天咱们就简单谈一下做长线和做短线的一些需要注意事项。对于长线而言，一般指的现货，很简单，逢大跌建仓，仓位建好后就尽量不要动了，对于大多数人而言，千万不要想着高抛低吸，就连经验丰富的交易员，也很容易在高抛低吸中被甩下车。对于短线来说，一定要勇于止损，很多将短线做成长线的都是没有止损造成的。若你做日内交易，就不要留隔夜仓，无论盈亏，都要在晚上休息之前将仓位清空，这样做最大的好处不是规避了夜间波动带来的风险，而是能够使你专心的休息，养足精神来面对第二天的交易，因交易而让自己身心疲惫的人是做不好交易的。另外，我们总是能看到这样一个现象，那就是做长线的与做长线的相互看不起，做长线的笑短线交易者每天交易却赚不到钱，做短线的笑长线交易者只会死拿……我的看法是，长线与短线的本质是一样的，只是思考的周期不同而已，两者没有高低之分，但如果你想学交易，那还是要从短线交易入手的，因为这样能够节省大量的时间成本，即使不做短线的也要学习一些短线思维，在牛市后期，你总能用的到。而做短线的呢，在牛市中一定要有长线仓位，这些仓位能保证你始终在车上，能对冲掉短线交易带来的不确定性风险。另外，无论做长线还是短线，技术分析都不是最重要的，最重要的是你要能够战胜你的内心。

2）BTC赶在双节前突破了20000点，开启了真正意义上的牛市，那么提前破20000点对后市走势有什么影响呢？我试着从我个人的理解谈一下。在我国传统金融圈有这样一个规律，如果节前大盘涨的好，那么节后大概率还会有一个快速上涨的机会，这其中的原因也很简单，春节是我国的传统节日，到时候我们要走亲访友，大家在一起聊天，很容易聊到“今年谁在股票上挣的多”这个话题，这就导致了很好的情绪散发，再加上大家都拿到了年终奖，有闲钱了，那么等到节后股市开盘，憋了很久的做多力量就会爆发。这里面有个条件，那就是特指我国的股市，至于比特币这个全球市场会不会也遵循这个规律，我并不清楚。但我们可以从历史中找经验，那就是每年的圣诞和元旦前，BTC大概率都会有一波下跌，因为大家有套现的需求，等到1月中旬，BTC的走势就会慢慢变好。为什么是1月中旬而不是农历春节后呢？我想同样是全球市场的原因，1月中旬正好是西方国家的“春节”之后，再加上BTC不休市，行情拐点就很容易发生在1月中旬这个节点。现在问题的答案出来了，那就是这次在节前突破20000点，带来的结果可能就是在2021年1月中旬后BTC大概率还会有一波加速上涨。现在又有问题了，那就是历史上双节前BTC大概率会回调，今年怎么没回调呢？我的理解是BTC已经回调了，但并没有回调到大家期待的点位上，比如16000点，也就是说，真正的牛市提前到来了。最后一个问题，短期内BTC还会大跌吗？如果你在20000点以前问我，我的答案可能是会，但鉴于目前BTC已经突破了20000点，我的答案就会有所不同，跌还是会跌的，但肯定不是20000之前的那种下跌，比如再想跌到20000之下，难了。当然，上面都是我个人的看法，对不对还有待验证，你看过后也要有自己的独立思考，更不能听了我的话就去梭哈， 你要做到大胆设想，谨慎决策。
