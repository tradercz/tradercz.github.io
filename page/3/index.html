2月11日比特币行情：走势很稳

发表于 2021-02-11 分类于 行情分析

过去24小时，整个市场的交易量2268亿美金，比前一天增加了100多亿，现在是北京时间2月11日的上午08点15分，BTC即时价格44550USDT，下面是具体分析。

1）BTC跌了，目前看只是杀多降费率，上升趋势没变。4小时与日线上看，比较舒服的回踩位置在42000-42500附近，也就是30日线附近，只要不跌破这里，这波上升趋势就不会被破坏。1小时上走弱，第一支撑43800附近，第二支撑42500附近。总结一下，安心过节吧，走势很稳的。

2）昨天的文章中我建议大家将大部分仓位都放在现货上，不要频繁操作，之所以让你这样做，一个重要的原因是到了牛市的中后期，暴涨暴跌会更加的频繁，而且还很难预测，若你重仓合约，不懂得风控，就很容易在某次大波动中“一朝回到解放前”。今天咱们接着这个话题，谈一谈如果要做合约都需要注意什么？一、拿一笔爆仓也不心疼的资金来做，只有不心疼，你才能有一个好心态，而好心态是交易成功的关键条件。二、每一笔交易都给自己设置一个极限点位，如果你要做多，那就在下面设置，比如BTC现在的价格在45000上方，4小时MA60在39000附近，你就可以将38000（MA60往下一点）设为BTC下跌的极限点。也就是说，哪怕BTC跌到了38000点，你仍不会爆仓。三、确保在极限位置的时候仍有资金补仓，而且补仓之后还留有剩余资金。也就是说你，你在38000补仓后，还有一部分资金没用完（防止更极端的行情发生）。三、不要频繁操作，频繁的止损、开仓和平仓只会让你的本金越来越少（止损是必要的，但咱们的前提是拿爆仓也不心疼的资金来做的），在趋势行情中，尽量用滚仓将利润最大化，关于滚仓，可以参考这篇文章。最后，你要明白的是，以上只是我的个人经验，可能并不适应于所有人。

1）BTC正式进入了机构喊单阶段，牛市下半程也正式拉开了帷幕。短期走势上，支撑压力已经无效，BTC逐渐疯狂，追高需谨慎。总结一下，短期目标差不多了，歇一歇会更好。

2）我们平时交易时，会遇到许多交易噪音，这些噪音往往会成为独立思考的绊脚石，下面我们一起来看看这些常见的交易噪音。一、微信群。这是交易噪音的主要来源，它的特点是传播速度快，一旦一则消息出来，就会迅速传播开。乍一看这是好事，但实则不然，币圈中大部分的微信群都提供不了太多有价值的东西（包括我们自己的微信群），反而会产生很多的交易噪音，如果你太容易相信别人的观点，那么就非常容易受到微信群的影响。二、微博。微博一般是大V发表观点的地方，但由于愿望思维和有限注意力偏差的存在，你很难对一个观点的对与错进行判断，很多人看微博，仅仅是为了给自己的观点找证据而已，也就是所谓的鸡汤。三、币圈APP内的社群。与微信群和微博一样，都是交易噪音的发生地，唯一的不同是这些社群还有一个其他目的，那就是为自家产生引流。看过我们『行为金融学』系列文章的同学都知道，人是懒于思考的动物，因为太费能量了，我们总喜欢拿来主义，或许这在其他行业是行的通的，但在做交易上，你必须打破这种魔咒，要学会独立思考，不要轻易地被一句话、一则消息改变你的计划（这也是我很少在微信群发声的原因），若你不这样做，市场会教你这样做。

2月8日比特币行情：短期关注这里

发表于 2021-02-08 分类于 行情分析

过去24小时，整个市场的交易量1713亿美金，比前一天略多，现在是北京时间2月8日的上午08点17分，BTC即时价格39026USDT，下面是具体分析。

1）昨天震荡了一天，大方向并没有变化，只要未来能在38000上方稳住，60日线就会上翘，那么日线级别的上升趋势就可能会被开启。4小时上，在MA30附近止跌，算是一次常规回调，1小时上，MA30已经下穿MA60，1小时周期趋势变弱，若不能迅速站上39500，则后面很有可能测试38000点。总结一下，大趋势是好的，但后面还有可能测试38000，短期关注39500点的得失吧。

2）今天谈谈我对金融市场短期走势的一些感悟，我认为市场短期走势有3个特征，我们分别来说。一、短期市场走势不存在因为A发生了B就一定会发生这样的规律，也就是说，影响短期市场走势的因素是很复杂的，单一因素对市场走势的影响不具备确定性和唯一性，也就是短期走势不可预测性。二、市场短期走势具有反直觉性，这里主要是指对散户反直觉，大部分散户认为短期市场会涨，那大概率就不会涨，大部分散户认为市场对跌，那大概率就不会跌。三、短期市场走势具有反身性。当认为短期会涨的人数足够多时，市场真的就会涨，当认为短期会跌的人足够多时，就是真的就会跌。我把以上3点称为金融市场短期走势具有量子态，也就是说，不管我们用什么方法，都无法准确预测市场的短期走势，这是一个动态的过程，每一秒都在发生变化（当然，这里使用量子态只是做形象说明，也有可能因我对量子理论理解的不正确而导致错用，只是在找到更合适的解释前，我们就暂时用量子态来描述了）。现在你知道了短期市场的这3个特点，就可以试着用它来解决最近大家比较关心的一个问题了，那就是CME的以太坊期货会对市场造成什么影响呢？那就是大概率没什么影响。

2）目前为止，我们已经学完了『行为金融学』的理论部分，并且在之前的文章中也提到了过度交易与卖哪个这两个问题，今天的重点是谈谈投资者在买入时容易犯哪些错误。一、喜欢频繁买卖同一个币，特别是之前盈利的币，而过度交易理论的结论是并不是交易的次数越多盈利就越多，相反的，交易次数越多，交易摩擦越大，结果反而跑不赢中长线策略。二、由于参考点依赖，很多人都喜欢越跌越买，而正确的做法应该是“向前看”，要有全局思维。三、追涨杀跌，这个问题在散户中普遍存在，本质也是“往回看”的问题。四、只喜欢在几个涨的不错的币中做选择，有限注意力偏差。以上4种行为都是大家在买入时容易犯的错，你现在可能有一种感觉，那就是这些有问题的操作都能用行为金融学来解释，难道行为金融学只会解释、就不能有点实际的用途吗？下面我就跟你来说一个我之前常用的策略，那就是买入前一天涨的不错的币，卖出已经连涨三天的币。第一天涨的不错的币已经在短时间内获得大量的关注，第二天大概率还会继续涨，连涨三天的币一般来说就该调整了。你看，这个策略的本质是预测投资者的错误并利用这些错误来盈利，而目前很多的量化交易策略，同样也是利用人们犯错的机会获利的。最后要提醒你，上面的策略或许会帮助你短期盈利，但若想要赚大钱，还是要靠趋势行情里的中长线策略。

2月5日比特币行情：小回调

发表于 2021-02-05 分类于 行情分析

过去24小时，整个市场的交易量1718亿美金，比前一天增加了约300亿，现在是北京时间2月5日的上午08点20分，BTC即时价格37105USDT，下面是具体分析。

1）BTC没有站上38500，目前正处于小周期的回调之中，要想保持多头趋势，这次回调不能跌破35000点。结构上，下方支撑36000、35000点，上方压力38500、40000点。从1小时周期来看，已经回调至MA60附近，如果是强势行情，到这里应该会止跌回升，反过来说，若跌破1小时MA60，则会测试35000点处的支撑。总结一下，目前看仅是小回调。

2）我们继续谈『行为金融学』，今天的重点是羊群效应。提起羊群效应这个词，你一定不陌生，它指的是人都有从众的心理和行为。你需要明白的是，羊群效应是一种社会现象，产生它的原因可能是理性的，也可能是非理性的，它是信息瀑布的一种（至于什么是信息瀑布，你可以回去看咱们2020年12月1日的文章）。下面我们分别举例说明羊群效应对金融市场的影响。关于币市中理性的羊群效应的例子，灰度基金是一个，当灰度注册某个币的基金后，人们就会更加关注这个币，就可能更加容易挖掘到该币的价值。那么非理性的羊群效应呢？会容易造成资产泡沫，特别是在牛市中， 乐观的预期促使散户蜂拥而至，很快便会将币价推升很高，一旦下跌，又会很快将币价快速下杀，造成期货爆仓，这也是牛市中的回调大都能在很短的时间内完成的原因。那么针对这两种羊群效应，我们应该如何做策略呢？一是要重点关注灰度币，二是在大家恐慌的布局，在别人乐观的时候谨慎追涨。关于第二点，我特别有感触，最近的每次回调，都有不少人来问我要不要卖，而上涨时，更多的人会来问能不能买，你看，这就是典型的羊群效应。殊不知正确的做法应该是在下跌期间布局，在上涨阶段持有和卖出。比如最近的这次回调，在很多人的心中，下跌幅度还是挺大的，于是就更容易卖出标的，而我给圈子中同学的建议以及我自己的实际操作都是在下跌期间持续布局，在上升阶段持币不动，伺机卖出。

2）我们继续谈『行为金融学』，今天的重点是狭隘框架理论。传统金融学认为，在做决策时，理性人的视野是全局的。在行为金融学下，我们的决策是会受框架影响的，简单地说，人决策时没有全局观就是狭隘框架。那么狭隘框架是如何影响我们的交易呢？最主要的影响有两个，一个是在有多个资产进行组合时，只关注单个资产的涨跌，另一个是频繁清点资产。如果你过分关注单个币的涨跌，就很可能将某个时间段内涨的比较好的几个币进行组合，但其实这不能叫组合，组合是要降低总资产的波动率的，而你是在增加波动率，你应该拿几个波动方向相反的币进行组合。当然，你要明白的是，由于币市有同涨同跌的特征，组合效应并没有股市强。那么频繁清点资产呢？我相信这几乎是所有散户正在做的事情，有人甚至一天多次清点资产，清点资产最大的影响是会大大增加参考点对投资决策的影响，因为你每清点一次资产就多了一个参考点，若你恰好在资产顶峰进行了清点，在之后一段时间内，你都会受“顶峰”的影响，盈利时你就更容易落袋为安（害怕利润继续回撤），亏损时你就更容易加大投资以摊薄成本（好不容易快回本了又跌了下去，心有不甘继续干）。那么我们应该如何做呢？还是那句话，只向前看，不向后看。举个例子，如果你以20U的价格买入了A，目标价是50U，现在涨到了45U，你该如何操作？还记得我们之前讲损失厌恶时说过不要看成本价吗？你应该这样想，现在价格是45U，目标价是50U，这点空间我可能只会拿1层仓位买入，好，那就卖出9层仓位，只剩1层仓位的A。现在，你明白了吗？

发表于 2021-02-03 分类于 行情分析

过去24小时，整个市场的交易量1511亿美金，比前一天略少，现在是北京时间2月3日的上午08点05分，BTC即时价格35413USDT，下面是具体分析。

1）历经将近1个月的回调盘整，BTC再次站上了30日线，若能站稳，下一波上涨趋势很快会开启。在这一过程中，其实4小时和小时级别更早的走了出来，你可以按照1月26日的文章中的方法来判断趋势的逆转。下方支撑35000附近，上方压力38000、40000以及前高点。总结一下，多头趋势初现。

2）我们继续谈『行为金融学』，今天的重点是心理账户理论。心理账户，是相对于真实账户而言的，人们喜欢按照不同的用途，将钱分成不同的账户。心理账户本身对我们的交易没什么影响，但它一旦与其他偏差比如损失厌恶结合起来，就会影响我们的决策。我举个例子，我们在交易时，总会不自觉的将实现盈亏与账面盈亏分成不同的账户，实现盈利时，大家一般就会变得能够承受风险，账面盈利时，却害怕利润回撤（损失厌恶），总想着落袋为安；实现亏损时，人就会变得更加的谨慎，不敢再轻易出手（很多人被312吓的好久不敢开单），帐面亏损时，由于不愿意认输，人就更喜爱风险，以至于不断的加码。再举个例子，现在有两个币，都很不错，但你的资金有限，只能选择一个，你选择了A，买入后发现A没有涨B却涨了，你就会非常难受，甚至比亏钱还难受，这就会导致你很可能在某一刻卖了A买入B，而这时你又发现B开始涨了，A却不涨了，于是你来回不停的折腾，最终两个币都错过了。你现在应该明白了，心理账户最大的影响是让你纠结于小细节，无法全局看待交易。如何克服呢？我们要尽量避免对单个账户或投资对象的单独决策，不要在意单个账户的盈亏，而应该将他们放在一起做全局考虑。另外，如果你仔细观察，就会发现生活中有许多有关心理账户的例子，比如面对某件事，你听到A对B说想开点吧，但B却不能释怀，这就是因为A与B将这件事放在不同心理账户了。除此之外，你还能举出其他有关心理账户的例子吗？

2月2日比特币行情：一个上行的契机

发表于 2021-02-02 分类于 行情分析

过去24小时，整个市场的交易量1584亿美金，比前一天增加了约200亿，现在是北京时间2月2日的上午08点11分，BTC即时价格33618USDT，下面是具体分析。

1）BTC的情况与昨天相比没有变化，依然受30日线的压制，现在的情况是只要向上轻轻一拉上升趋势就形成了，但若要跌破29000点，则后市就可能应该更加长久的调整。既然BTC没有太多可说的，那么其他币呢？我们今天来说说ETH，这是目前走势最好看的老牌主流币，也是最能影响BTC走势的币种，ETH后市大概率会走一波上升趋势，那么就很有可能带动BTC上行。总结一下，ETH的强势可能会带动BTC上行。

2）我们继续来谈『行为金融学』，今天的重点是决策权重理论。理性人认为，人们在做决策是会根据事件发生的概率选择最优解，但行为金融学认为，人在对概率的判断上会出错，具体表现在——低概率事件被高估，低概率事件被低估，高概率事件被低估，高概率事件被低估。我们分别谈谈这几种情况，第一种，低概率事件被高估最典型的例子就是买彩票，彩票中奖本来是概率很低的一件事，但中奖会给你留下深刻的印象，这时候你就会高估中奖发生的概率。在币圈，我将那些价格已经大幅低于历史最高价且喜欢突然诈尸的币称为彩票型代币，比如XRP，我一般是不会碰这种币的，但我知道有很多人就是喜欢这种币，赌这种币的突然拉升就是高估低概率事件。当然，我并不是说买XRP的人就是错的，比如你有一种方法能抓住它大多次的上涨，那也是一种相当厉害的策略。第二种，低概率事件被低估，在牛市中，一些小的不足以影响趋势的事件发生时，往往会被人们忽略，但随着此类事件的增多，趋势会悄悄的被改变，但由于我们总是将它们改变趋势的概率视为零，就会导致趋势真正的改变时也察觉不到，这种后知后觉就是典型的低概率事件被低估的案例。第三种，高概率事件被低估，在币市做合约，被爆仓的概率是很高的，但你总是觉得不会爆仓，也就不设止损，这就是高概率事件被低估的现象。第四种，高概率事件被高估，比如对趋势的判断中，我们需要同时满足几个条件后市才会涨，而当这几个条件单独发生的概率都在90％以上时，我们就会得出后市会涨的结论，这其中的原因就是我们将这个90%以上的概率当成100％，也就是说，在需要同时满足多个条件的事件中，最容易高估其发生的概率。以上就是决策权重理论，它与前两篇中的参考点依赖、在盈利与亏损时对待风险的不同态度组成了行为金融学中第一个诺奖理论——前景理论。
