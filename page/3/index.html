11月6日比特币行情：没有见顶迹象

发表于 2020-11-06 分类于 行情分析 阅读次数： Valine：

1）站稳14000点后，BTC一路拉升，24小时内拉升了近1500刀，在这种强势行情下，大部分指标都失灵了，这也给技术分析带来了困难，对此，我们应该怎么办呢？我建议大家记住以下几点：一、如果日线级别出现冲高回落，并且伴随着巨量，那么就要考虑行情到头的可能。二、观察BitMex、OKEx等头部交易所的BTC永续合约的资金费率，如果处于0.03％长达数天，则也是行情终结的可能。三、观察各大群里散户的情绪，包括微博大V们的情绪调查，如果大多数散户都看多，那么行情也走不远。根据以上指标，我们可以得出目前的行情并没有见顶迹象，也建议大家不要轻易猜顶。
