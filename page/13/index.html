11月7日比特币行情：盘整后仍有新高的可能

发表于 2020-11-07 分类于 行情分析

过去24小时，整个市场的交易量1404亿美金，比前一天略多，现在是北京时间11月7日的上午08点21分，BTC即时价格15655USDT，下面是具体分析。

1）昨日冲至15960点后开始回落，最低跌至15166点，在这一过程中，各大交易所永续合约资金费率开始走高，后又在下跌过程中有所回落，除了币安的USD永续合约外，其他交易所的费率还算是处于相对合理的水平。行情方面，昨天的成交量虽然不小，但振幅却不是很大，后市可能继续在15166-15960区间震荡，在此过程中，只要能给散户的追高热情降降温，后市大概率会继续走高。总结一下，行情没有见顶，短期主要看15166-15960区间震荡吧。还有，只要回调不跌破14000，则15960大概率不会是年内高点。

2）在平时的操作中，你是否有过挂单只差一点点而没有成交的遗憾么？今天咱们来谈谈分批操作的重要性。找到压力与支撑后，我们要在压力与支撑附近挂单，对于新手来说，往往挂一单就不管了，有时候能顺利吃到，而更多的时候是吃不到的，这时候就推荐用分批挂单。比如15000是最近的支撑位，你可以分别在15150、15100、15050、15000、14950、14900等位置挂单，如果全部成交，平均价格就是15000。当然，更多的时候是只成交了一部分，比如10150-15000之间的挂单成交了，更下方的没有成交，这样总成本就没有只在15000挂单划算了，如何解决呢？可以采用金字塔的挂单方式，如15150挂10张，15100挂15张，15050挂20张，15000挂25张这样，虽然这样做的成本还是没有只在15000挂单显得划算，但在增加了订单的成功率后还尽可能的将成本控制的最低，已经算是很好的策略了。总得来说，交易向来都是不完美的，我们只是尽可能的让交易符合我们的预期而已。另外，分批挂单背后的思想还是仓位管理，你发现了吗？

发表于 2020-11-06 分类于 行情分析

1）站稳14000点后，BTC一路拉升，24小时内拉升了近1500刀，在这种强势行情下，大部分指标都失灵了，这也给技术分析带来了困难，对此，我们应该怎么办呢？我建议大家记住以下几点：一、如果日线级别出现冲高回落，并且伴随着巨量，那么就要考虑行情到头的可能。二、观察BitMex、OKEx等头部交易所的BTC永续合约的资金费率，如果处于0.03％长达数天，则也是行情终结的可能。三、观察各大群里散户的情绪，包括微博大V们的情绪调查，如果大多数散户都看多，那么行情也走不远。根据以上指标，我们可以得出目前的行情并没有见顶迹象，也建议大家不要轻易猜顶。当然，越是到了行情末端走的也会越快，现在没有迹象不代表几天甚至几个小时后没有迹象，我们所做的就是耐心等待顶部信号的出现。

2）在我们回过头去看之前的行情时，就会有一种上帝视角的感觉，还有很多时候会觉得自己当时看对了，结果该赚的钱没赚到。我们仔细思考一下，这真的是你应该赚却没赚的钱吗？不一定吧。或许这只是你当时猜对了呢？就算你不是猜的，就一定能保证你会买入吗？就算买入了，就能保证你会坚持持有吗？你能保证在大波动面前没有被甩下车的风险吗？这就好比你经常听到某些人说，我要是早几年知道BTC就好了，仔细想一想，早几年知道了你就一定能持有到现在吗？16年以前BTC是骗局的观点是非常流行的，你能保证在那种环境中还能客观思考吗？所以你看，交易不止是技术分析，更是心理的博弈，在成为优秀的交易员的路上，技术瓶颈好突破，心理这一关却挡住了大多数人，这也是交易最难的地方。因此，如果你始终无法在交易中盈利，很有可能不是技术的问题，而是心理的原因，你应该多抽时间学习交易心理方面的知识。

11月5日比特币行情：多头继续强势

发表于 2020-11-05 分类于 行情分析

过去24小时，整个市场的交易量1023亿美金，比前一天增加了200多亿，现在是北京时间11月5日的上午08点30分，BTC即时价格14146USDT，下面是具体分析。

1）昨天日内最低跌至13525点，后遇到强势的买盘，早上8点日线收阳，说明多头趋势很强，后市继续走高的可能性很大。结构上，已经成功突破了14000点，接下来的目标就是15000、16000这些整数关卡，进而上攻17000附近的压力位。日内走势上，有回调到14000下方的机会。总结一下，白天可能会走震荡，晚间更可能创新高。

2）你可能经常听别人说顺大逆小，顺大逆小是做交易的一种方法，意思是说我们在大方向上选择与市场一致的方向，但是选择一个小的回调逆势介入。比如市场的大方向是向上的，我们就做多，但我们不追多，而是找回调的时候再介入，这样做最直接的好处是能够较好的控制成本。你现在听我一说，马上就明白了，不就是顺大逆小嘛，很简单啊。但我们在实际操作中，不仅不容易做到顺大逆小，还很容易做反，也就是总是喜欢顺小逆大。比如BTC经过一段很好的上涨后开启了回调，按照顺大逆小的做法，现在是加仓的时候，但你会更容易地减仓。你看到BTC涨的很好，刚开始还告诉自己等回调再入，可是一直没有回调，接下来你就更容易去追涨。为什么会这样呢？主要是由于我们太关注短期的涨跌而忽略了长期的趋势造成的。除此之外，还有一个心理上的因素，那就是我们总是喜欢那些一买入就盈利的标的，甚至承受不住哪怕只有短短几天时间的亏损期。所以你看，做交易最难的往往不是技术，而是心理，很多人之所以做不好交易，不是技术不行，而是难以说服自己的内心。

11月4日比特币行情：下一目标17000

发表于 2020-11-04 分类于 行情分析

过去24小时，整个市场的交易量876亿美金，比前一天略少，现在是北京时间11月4日的上午08点23分，BTC即时价格14003USDT，下面是具体分析。

1）经历小波折后，BTC还是向14000点发起了冲击，虽然现在还没有站稳，但多军已是胜利在望，拿下14000点后，下一个目标就是17000点附近，这是18年1月的高点。需要注意的是，17000并不一定能达到，但趋势是这样的。我们可以用两军打仗来形容，多军为进攻方，空军为防守方，14000这座城被拿下后，多军就会向17000城市进攻，至于能不能拿下，中间会经历空军的哪些阻挠，咱们并不清楚，但可以确定的一定是，这中间的阻挠力量一定没有17000城市的力量大。短线方面，下方最近的一个支撑位在13800-13940区间，不破这里，短期看涨。另外，美国大选今天下午会有结果，也可能存在变数，届时会不会对金融市场造成影响，咱们拭目以待。

2）你可能经常听到交易体系或者交易框架这个词，也多多少少了解一点，但却没有一个完整的认识，今天咱们就来谈谈这个话题。所谓的交易框架，其实说白了只有3句话——买什么、怎么买以及买了之后怎么办。先说说买什么这个问题，新手最常犯的错误的什么都想买，今天听到这个币有利好了就买一点，明天又听别人说那个币不错也买一点，仿佛不买就会错过百倍币似的。 这里面有两个错误，一是标的选的太多，二是总是被消息牵着鼻子走。其实选择买什么的过程就是定位自身投资风格的过程，你善于发现别人没有发现的机会，就可能是一个价值投资者，你对图形、数据以及市场情绪比较敏感，就可能是一个趋势交易者，你善于比别人早一点发现下一个热点，就可能是一个很好的投机者。总之，你是什么样的人？你擅长什么样的交易方式？是短线还是长线？是技术分析还是价值投资？你要结合你自身的特点选择买什么，但不能被消息牵着鼻子走。接着说说怎么买？是逢低买入？还是突破后去追？逢低买入的成本低，但不确定性大，突破去追成本高但不确定性要小一些。又或者是利用定投摊薄成本？还是盈利后使用杠杆实现浮盈加仓？你看，不同的风险偏好有不同的交易风格，低风险偏好的人就不适合高风险的交易方式。最后是买了后怎么做？对于这个问题，我们坚持买卖同一性就行了，也就是说，如果买入理由不成立了，那就卖出。比如上升趋势形成后你开始买入那就在上升趋势结束后卖出。再比如你对某个币不了解，但你听你朋友的建议买入了，后来他又建议你卖出，你也应该卖出，而不应该因为看到有利好消息而不卖了。这就是买卖同一性原则，也是职业交易员需要遵守的一个基本原则。你看，你在解决买什么、怎么买和买了后怎么办这三个问题的同时，其实就是打造自己的交易框架的过程。需要注意的是，由于每个人的自身特点不同，对市场的感知也不同，交易框架也就可能千差万别，有人喜欢买入BTC进行长线投资，而有人就会觉得BTC就是投机品，要快进快出，毕竟我们不可能让一个急性子的人拿着BTC等上个3、5年再卖出，这不现实。对于我们的VIP同学，你看到的是我个人的交易框架，虽然我自认为我的框架适合大部分新手，但却不一定适合你，这就要求你要在我的基础上打造你自己的交易框架，这是你一定要明白的。

1）BTC从昨天下午4点开始放量下跌，最低跌至13195点，随后遇到一波力度不小的抄底资金将BTC拉回13300上方，那么现在的行情如何了？结构上，回到了12800-14100区间，其中12800是10月21日收线的位置，如无特别的利空，跌破这里的概率很小。小结构上，13195（昨日低点）-14100区间。日内走势方面，可能在4小时MA60进行震荡，若4小时MA60不破，则后市仍会继续走高。此外，昨天那则火币的消息更是增加了散户的恐慌情绪，要知道在恐慌情绪下散户是最容易交出筹码的，如果你做的是长线，是不应该在这个时候下车的，从夜间的拉升也能看出来，国外资本一直在买买买，而我们散户却在抛售，孰高孰低，可见一斑。总结一下，止跌了，日内就看它在4小时MA60上方震荡吧。还有，昨天说主流币开始表演了，结果被打脸了。

2）昨天BTC下跌时，有同学比较恐慌，现在说一下我对此类情形的看法。一、BTC一路拉升，基本没有像样的大回调，即使回调也是24小时内就完成了，像昨天这种回调力度，根本不算大，哪怕跌到12000点，日线级别上升趋势依然没有被破坏。二、对于拿着现货长线和合约趋势单的，只要合约单的强平线足够低，比如在12000以下（距离现价还很远），是可以不动的。这也是我们说过的右侧交易法，这些长线单可以等顶部出现了再操作，你也可以思考一下这样做的好处及坏处。三、如果有一天一下子跌了1000刀，你也不必害怕，而且牛市中这种级别的回调一定会发生。四，对于短线交单的，跌破13600（现货）就要考虑止损了，这点策略版分析中有提到。另外，再谈谈为什么你总是拿不住筹码。我认为有以下几个原因。一、对趋势理解的不够深，只要一跌就慌，不知道能跌到哪里，这属于技术知识，只能在学习与长期的市场观察中历练才行。二、你关注的周期太小了，就很容易在这种很常见的回调中丢掉筹码。三、仓位不合适。曾经有人说过，理想的交易状态是手心微微冒汗但睡得香，手心微微冒汗，说明在你心中仓位已经足够了，睡得香，说明你对当前的行情有信心。如果你建仓了后睡不好，那就说明这样的仓位不适合你或者你对当前的行情没有信心，你之所以这样做就是想赌一把，我建议你减仓。顺便说一下，我大部分的时间都是处于睡得香的状态，这不是因为我睡眠好，而是我的仓位让我很舒服。总之，做交易经验非常重要，如果你在交易上有困惑，不要着急，也不要想着找捷径，唯一的办法是多学多练。

10月30日比特币行情：这波调整的时间可能会少于我们的预期

发表于 2020-10-30 分类于 行情分析

过去24小时，整个市场的交易量1411亿美金，比前一天增加了300多亿，现在是北京时间10月30日的上午08点35分，BTC即时价格13431USDT，下面是具体分析。

1）昨天BTC走了个小反弹，量能中等偏大，昨晚20点再次往前低12888方向测试，在13000下方遇到明显的买盘支撑，初步确认12888就是这次回调的低点了。接下来BTC可能会将13300点（4小时MA30附近）做为短期支撑继续震荡，大家可以将13500做为日内观察的重点，如果今天BTC成功站上了13500点甚至突破了13600点，那么后市大概率会继续走高，届时这波调整的时间将会大大少于我们的预期。总结一下，这波调整的时间可能会变短，且后市仍有走高的潜力。

2）我们先来看一个有意思的实验，这个实验是这样的：有两组人，A组是由于没有时间而不得不将孩子送到幼儿园的家长，B组是有时间而不需要将孩子送到幼儿园的家长，我们将一篇孩子在幼儿园和在家里成长好坏对比的论文分别拿给A和B组的家长看，但这篇论文的结论是非常模糊的，怎么解读都可以。结果A组的家长认为这篇论文的观点是孩子应该送去幼儿园，而B组的家长则认为论文的结论是孩子待在家里最好。为什么A和B组的家长会得出截然不同的结论呢？这就引出了我们今天的主题，那就是人们都是有愿望思维的。什么是愿望思维呢？愿望思维是一个人为了坚持自己的信念而无意识的去解释事实的思维现象。需要注意的是它是无意识的，它会让人选择性地接收那些符合个人愿望的事实，把事实往符合愿望的方向上解读，导致决策和行动的偏差。举个现实中的例子，某个国家与咱们比较友好，那么在很多网友心中就会有这样一个观点，那就是不管这个国家对我们做了什么，哪怕是伤害我们的事情，这些网友也总会说这是我们国家有意安排的。再比如一个男生追求某个女生被拒绝了，这个男生就这么认为，其实她是喜欢我的，只是犹豫害羞（或某种原因）拒绝了我而已。你看，愿望思维其实就是一厢情愿。在币圈，愿望思维也随处可见，其中最典型的是一篇多空不明的文章或新闻在多空双方眼中分别会拥有明确的多空观点，也就是说，多头看什么文章都觉得有利于BTC，而空头看任何文章都认为不利于BTC，更可怕的是，他们还很难发觉这种偏差。你仔细回想一下自己有没有相似的经历呢？我想大多数人都是有的吧。那么如何纠正呢？我们在做判断的时候一定要对愿望思维保持警觉，要知道世界上的事情不是以你为中心展开的，事情总是几乎随机地发生，不会总往你合意的方向走。人生不如意事十常八九才是正常的，当你做出一个判断的时候，如果这个判断正好顺从了你的情绪，那你就应该反问自己了——这会不会是愿望思维呢？

10月29日比特币行情：低点可能已经出现了

发表于 2020-10-29 分类于 行情分析

过去24小时，整个市场的交易量1031亿美金，比前一天增加了约100亿，现在是北京时间10月29日的上午09点16分，BTC即时价格13248USDT，下面是具体分析。

1）BTC终于在距离14000不到150美金的地方开启了回调，连续6个小时的放量下跌释放了不少获利盘，那么如何看待这次回调？我们将10月19日20点的那根4小时K线作为起点画一条斐波纳契回撤线，发现昨日低点12888是在0.382附近的（在强势行情中，如果发生回调0.382就是可能的目标位），又加上昨天20点的那根4小时K线带着长长的尾巴，说明有不少的人在这里进行了买入或者平空操作，那么昨日低点就可能是这次回调的目标位，那么接下来我们主要关注12888的支撑情况即可，后市很可能会对12888进行再次的测试，如果不破，则说明回调结束，后市看涨。当然，也要做好跌破12888测试12400-12500的准备。总结一下，12888是有可能成为这次下跌的低点的，如果不是，当我没说。

2）今天我要解答一个在币圈经常被提及却很少有人解答的问题，那就是“如果你交易就能挣钱，为何还要做付费社群呢？”，在很多人看来，这就是一个悖论，还有一丝做收费社群的都是不可信的味道。要回答这个问题，我们首先得明白交易的本质是什么？那么什么是交易的本质呢？我给你做个比喻：一、做交易就像是从一个看不见内部的箱子里拿钱，当然，这个箱子里不仅有钱，还有一条蛇，你得用手去拿，还要尽量的靠你的经验避开这条蛇。二、新手在伸手拿钱的时候往往都会带有一些侥幸心理，他们总觉得自己能够顺利拿到钱还不会被蛇咬。三、老手在伸手拿钱的时候会根据经验感知蛇周围的温度，一旦认为蛇在手附近就会赶紧停止放弃拿钱这个动作。你仔细想一下，交易与伸手拿钱这个动作是不是很像？新手交易者总是认为自己很幸运，而老手由于见识过市场的残酷而有些胆小甚微。说了这么多，交易的本质是什么呢？就是交易是充满不确定性的。我们可以通过成一些科学的方法来降低不确定性，而做收费社群就是降低不确定性的途径之一。除了币圈，传统金融圈也有同样的例子，比如优秀的交易员会写书，会做讲师，会开资产管理公司，都是将不确定性变成确定性的例子。说到了这里，我还想继续延伸一下，那就是有了交易框架可以保证稳定盈利，但你能不能赚大钱却是由你的运气、胆识决定的，这就像北电一个班只能出一两个大明星一样，其实同班同学的资质、长相各方向都差不多的，为何只有他或者她能红呢？这也是与一个人的运气有关。这和“他凭什么能赚大钱”是一样的，因为他的运气比你好，但你的运气又比那些破产的人好很多，只是由于幸存者偏差存在，你已经很难记起那些失败者而已。说了这么多，只是想要告诉大家，技术可以保证你稳定盈利，但却不能保证你赚大钱，但如果你想要赚大钱，一定得有保证不亏钱的技术才行。最后，关于付费社群再多说一句，那就是币圈不乏通过收费社群骗人的，因为做社群不需要资质，任何人都可以做，再加上币圈投资者辨识能力不是很强，那些在传统金融圈有忽悠经验的人到了这里同样很吃香，因此，大家平时一定要警惕，记住天上没有馅饼，即使有也不会掉到你手里，只有没有侥幸与暴富的心理，就很难上当受骗。
