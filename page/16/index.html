10月8日比特币行情：比耐心的时候到了

发表于 2020-10-08 分类于 行情分析

过去24小时，整个市场的交易量774亿美金，比前一天减少了约200亿，现在是北京时间10月8日的上午08点58分，BTC即时价格10640USDT，下面是具体分析。

1）如昨天所说，震荡幅度越来越小，波动率也降到了极点，如果这种情况下选择方向，那至少会是千刀级别的大波动，我们要做的就是耐心等待即可。月线上支持跌到9200附近，也就是说，只要不破9200，大周期还是向上的。周线上不能跌破9800，如果跌破则大概率会去9200附近。日线上下不破10500，上不破10780视为震荡，下破10500或上破10780视为方向选择的开始。另外，如果你看月线，这两个月的行情也只是一次小小的回调而已，但这已经令很多人感受到“熊市”的残酷了，因此，别人口中的牛熊市可能与你自己认为的不一样，你最好先找准自己的周期再进行行情的判断，这样才不容易产生误解。总结一下，这个位置可上可下，耐心等待方向的选择吧。

2）昨天我们讲了复盘，今天我们再进一步，来谈谈重启。我们每次复盘都能让自己有所成长，今天的我们相对于昨天来说是全新的，在开启一次新的交易时，我们抛去昨日的所有损失和荣耀，抛去了沉没成本与负担，这样才能使得这次交易更加纯粹。要知道，大多数人都会面临两个挑战，一是舍不得旧的，二是无法开启新的。如果你无法做到重启，一次操作的失误就很可能会影响后面一系列的决策，从而使你的交易系统彻底瘫痪，这对于交易员来说是最致命的打击，很可能他的交易生涯也会随之结束。那么如何做到重启呢？在每一次新的交易时，你都要问自己昨天的亏损有没有对这次交易造成影响？这次交易是为了翻盘吗？又或者昨天的盈利有没有对这次交易造成影响？（因为昨天盈利了）所以这次亏损也无所谓吗？如果你的答案是肯定的，那你这次操作就不是重启，而是前一次操作的延续，你要想办法将你的答案变成否定后再开启新的交易。值得注意的是，重启不仅对交易有用，对你人生的方方面面也都同样有用，我们的生命很短暂，根本来不及为昨日哀叹，只有学会不断的重启，才能做到不畏将来，勇往直前。

1）昨天放量下跌，几乎将前两天反弹的涨幅全部跌没，但行情并未发生变化，仍然属于震荡行情，并且现在正处于日线震荡大三角的末端，最近几天大概率会选择方向，下方关注10450-10500的支撑情况，上方关注10750-10800的突破情况，也就是说，只要跌破10450或者突破10800，则都可能是一段下行趋势或上行趋势的开始。总结一下，再耐心等待几天，等待方向的选择。

2）我们今天的主题是复盘。为什么要复盘？咱们先从人脑的构造说起。我们在7月19日的文章中说过，人脑中的杏仁体控制着我们情感，前额叶皮层控制着我们逻辑思维，杏仁体与前额叶皮层最大的区别是，杏仁体反应快，不太准，有时候会犯错，但是不费能量；前额叶皮层的反应相对来讲比较慢，又非常费能量。这也就是说，有两套系统交替控制着我们的日常行为，我们把这两套系统分别称为有意识模式和无意识模式，有意识模式就像新手司机一样，做每一个动作之前都要先在脑中预演一遍，而无意识模式就像老司机，能很快的换档、掉头、刹车等，然而现在的问题是，这两套系统总是错位，比如一个男士为了买一个2000元的手机能思考半天，而买卖几万块的BTC却不用脑子，一个女士选个包包能跑一上午，而选老公却只能了三分钟。另外，我们经常强调不要冲动交易，但其实这是很反人性的，真正能做到的人并不多，这时候复盘就显得很重要了，你可以趁着晚上睡前的一段时间回顾一下当天或近期的交易，看看逻辑上有没有问题，如果有问题，就得及时纠正了，记住犯错不可怕，可怕的是犯错后不改正而导致错误越来越严重，这才是真正需要大家警惕的地方。

10月4日比特币行情：严重缩量，没有方向

发表于 2020-10-04 分类于 行情分析

过去24小时，整个市场的交易量666亿美金，比前一天减少了300多亿，现在是北京时间10月4日的上午08点38分，BTC即时价格10544USDT，下面是具体分析。

1）目前行情遇到的两个问题，一是严重缩量，二是继续受30日线的压制。严重缩量代表没有方向，后市上涨和下跌都有同样的可能性。受30日线压制说明大盘无力，偏空，但就像昨天分析里所说的那样，只要还在10400上方运行，就不要看的太空。总结一下，总体行情与昨日一样，并无变化。

2）在交易时，大家都应该有一点防爆思维，因为它优先于所有的交易技巧。什么是防爆思维？就是无论爆仓的可能性有多小，你都要对可能的爆仓做出充足的准备。我给你讲一个美国长期资本的例子吧。这家公司的掌门人是梅里韦瑟，他被誉为华尔街债券套利之父。除此之外，团队里有两位诺贝尔经济学奖得主，有美国财政部前副部长以及美联储副主席，还有所罗门兄弟债券交易主管……我现在要说这是一家由世界上最聪明的人组成的团队，你不会反对吧。与团队相匹配的是这家公司的业绩，前几年投资回报率分别是：28.5%、42.8%、40.8%，对于一家基金公司来说，这可是相当辉煌的成绩，但就是这样豪华团队组成的公司，在1998年的数周内损失了90%以上的资产，原因是它遇到了俄罗斯债务危机，而长期资本使用的杠杆非常大，仓位又极重，当市场剧烈波动时就爆仓了。这就是小概率事件导致大损失的一个很好的例子，它告诉我们，无论做什么交易，哪怕看起来毫无风险也不要AII in（梭哈），同时要做好随时撤退的准备，这也是防爆思维的精髓。有同学说了，反正我现在也没有多少钱，爆仓也不怕，再说了，不搏一搏怎么单车变摩托呢？我告诉你，你每搏一次就相当于赌一次，而赌博是会上瘾的，钱少的时候你不养成好习惯，等钱多了你还会犯同样的错误的，记住，在交易的世界里，不是胜者为王，而是剩者为王。另外，防爆思维不仅适用于交易，还适用于你人生的方方面面，比如健康，你认为你现在还年轻就熬夜、抽烟、酗酒，其实这也是在用的的健康AII in，如果你不及时停下来，将来你的健康也很可能爆仓，所以，从现在起，放下不健康的生活方式，从不起眼的生活习惯就开始锻炼，因为生活和投资一样，都是一件需要长期坚持的事。

2）你可能常听到“会买的是徒弟会卖的是师傅”这句话，从中可以看出，卖比买更难，具体表现如下：新手往往在有了一点利润之后就会立刻卖出，而对于有了一些交易经验的同学（会点技术又懂点基本面），往往在该卖的时候没有卖，白白坐了一次过山车。如何解决这个问题呢？其实也很简单，就是只做纯粹的技术派。什么意思呢？你不用考虑外界消息、基本面等因素，只根据图标做买卖决策。如果你做趋势交易，只要跌破趋势线就开始卖，突破趋势线就开始买，在这一过程中，你不用考虑其他因素，也不要怕做错，做错了止损即可。有同学注意到了，我在上面用的词是“开始卖”和“开始买”，而没有直接用“卖”和“买”，这是让大家养成分批操作的习惯，即使你再有把握，也要一点一点的买卖，比如你可以在突破趋势线的时候买一点，突破颈线的时候再买一点，突破前高再买一点。此外，关于买卖的时机还得考虑一个重要因素，那就是你的交易周期，有同学做长线，那你重点参考日线就好，有同学做中线，那你就重点参考4小时，做短线的可以参考1小时或15分钟，总之，在开始买卖之前，你得弄清楚你的交易周期，这也是我不建议大家都跟着我操作的一个重要原因，我的建议和操作你可以借鉴，但你要操作时，一定得考虑到你的具体情况才行。

9月30日比特币行情：反弹可能继续但上升趋势仍未形成

发表于 2020-09-30 分类于 行情分析

过去24小时，整个市场的交易量829亿美金，比前一天减少了约100亿，现在是北京时间9月30日的上午08点49分，BTC即时价格10824USDT，下面是具体分析。

1）昨天早上7点一波快速下跌后开启了一小波反弹，到今天凌晨1点再次对一小时的低点10626进行测试，测试有效后继续反弹，截止发文时已经触碰到昨天的下跌起点10867，也就是说，中间这24个小时行情没怎么变，转了一圈又回到了起点。后市走势方面仍坚持昨天的观点，大概率会继续反弹，反弹的目标位11100-11200之间，若能突破11200，则很可能开始新一段的上升趋势。总结一下，目前仍处于9800-11200这个震荡区间，反弹可能继续，但上升趋势仍未形成。

2）今天的关键词是“慢就是快”。很多人特别是此前亏损严重的朋友都有快速翻盘的心理，由于他们亏掉了不少钱，就幻想着通过高人指导、高倍合约、赌翻倍币等投机的方式来实现财富的快速增值，殊不知这恰恰会使自己步入“病急乱投医”的地步。先说高人指导，其实这里面大都不是什么高人，很多人的水平其实只比你高了那么一点点，他们在你面前伪装成高人只是想让你付费而已，真正想要传授你技巧的人非常少。如果你有加入付费社群的经验，那你大概率会有这样的经历——入社群之前觉得老师很牛，进入之后却没怎么挣钱。究其原因，一是你的期望值太高了，你就是抱着投机的心态来的。二是他的方法不适合你。三是老师没有特别牛的能力。再说高倍合约，如果你运气足够好，可能一开始会盈利颇丰，但只要将时间拉长，你一定会输的很惨，因为目前市场上做高倍合约的人中有99.9％的都不能保证稳定盈利。最后是赌翻倍币，很多人乐此不倦，但为什么很少人能够抓住翻倍币呢？因为翻倍币本身的波动率都很大，在翻倍之前他可能先跌个50％、60％，你买入后很难守得住，特别是那些重仓买入的，一个急跌就会让你寝食难安，这时候只求不亏就行，还想什么翻倍呢？因此一旦亏损被抹平你很可能就卖出了，后面翻倍的机会也就再与你无关。所以你看，这个世界上并没有一种“药”能让你快速变富，如果有的话，你也得先花费大量的时间使自己有承受财富的能力，只有这样财富才有可能主动来找你，否则即使你侥幸抓到了一波，那结果也很可能是黄粱一梦。因此，在接受财富之前，你应该先有一套适合自己的交易策略和财富理念，不能着急，一定要理解“慢就是快”的含义，让时间陪你慢慢变富。

9月27日比特币行情：震荡行情，多空不明

发表于 2020-09-27 分类于 行情分析

过去24小时，整个市场的交易量801亿美金，比前一天减少了100多亿，现在是北京时间9月27日的上午08点26分，BTC即时价格10746USDT，下面是具体分析。

1）昨天缩量震荡，没有方面。细节方面，BTC重新站上了30日线，但行情并没有变化，一来30日线已经掉头向下，二来30日线并不是严格意义上的压力支撑位，它只是我们判断趋势的一个依据，也就是说，后面还可能围绕着30日线做震荡，只有长时间（比如一周）运行在30日线之上，进而站上60日线，中期趋势才可能抬头往上走。结构上，仍处于9800-11200区间，10500之上多头占优，10500之下空头占优。压力支撑方面，下方支撑10550、10136、9825，上方压制10850、10950、11200。总结一下，周末交投清淡，震荡行情，多空不明，耐心等待。

2）今天咱们来谈一个词——向死而生，这是德国哲学家海德格尔《存在与时间》中的一个概念，在海德格尔的眼中，死亡是两个词，一个是“死”，它指的是死的过程，我们每个人都正在走向死亡，而“亡”才是真正的死亡。向死而生说的是人只有面对死亡的威胁才会认真看待自己的人生，才会选择一个自己所要的、能够负责的人生。那么这句话与交易有什么关系呢？如果你深入思考了，就会发现人生与交易其实是一回事。人拿着父母给的一切来到世间，经历成长、欢笑、成功、失败、病痛、死亡这几个阶段，而交易是你拿着本金进入到交易的世界，历经懵懂、天真、快乐、懊恼等阶段最终到一无所有。换句话说，你现在所有的盈利都是暂时的，交易的终点就是亏光。有同学不理解，我做交易是为了挣钱的，怎么会亏光呢？我告诉你，你认为交易会挣钱是因为你只看到了某人的某个阶段，如果将时间线拉长，我们来交易的每个人都会亏的一无所有。如果你理解了，那向死而生就能用在交易上了——我们来交易总是要亏完的，何不趁着盈利的时候、还没有亏完的时候、有能力的时候多做一些其他的事情呢？比如该止盈的时候坚决止盈，还止损的时候坚决止损，该学习提高的时候一定要学习提高，该休息的时候一定要休息……我们要做不是要挣多少钱，而是要想着通过什么办法尽量的将我们的交易之路拉长，长到接近甚至超过我们的生命本身。有同学又会问了，如果我在赚了一大笔钱之后就离开这个市场了呢？我告诉你，如果真的是这样的话，你就是将你的交易之路无限期的冻结住，去好好的实现你人生中的其他理想了，你很好的饯行了向死而生这一格言，恭喜你。今天的内容过于偏哲学，对于没有接触过这一学科的人来说不太好理解，甚至会觉得有些扯淡，没关系，不理解的暂时先搁置一下，如果你做交易的时间足够长，我该保证早晚有一天你会理解的，在我心中，交易哲学比交易方法更有用。

9月26日比特币行情：不要过于担心市场会大跌

发表于 2020-09-26 分类于 行情分析

过去24小时，整个市场的交易量921亿美金，几乎与前一天持平，现在是北京时间9月26日的上午08点47分，BTC即时价格10658USDT，下面是具体分析。

1）经过前一天的拉升后，昨天缩量回踩，属于正常的市场表现，行情方面没有变化，上方仍然受30日线的压制，突破30日线后才会向60日线发起冲击，因此，想要向上突破还是比较有难度的。下方需要注意的地方有两处，一是10136，二是9825，这两处都是日线的低点，只要还在之上运行，震荡结构就不会被打破。另外，距离上次跌破30日线和60日线已经3个星期了，如果处于熊市，早就该偏离60日线并大幅向下破位了，很显然这种情况并没有发生，说明大家对于后市仍然很乐观，从这点来看，市场可能并不具备大跌的条件，既然这样，那咱们就不要过分的看空或看多，耐心等待市场的方向吧。

2）交易时如果能找到合适点位固然重要，但由于短期行情的不确定性，哪怕你技术再高，也很难准确的判断出一次拉升或回调的具体点位，即使判断对了，你当时也很有可能因（比如不确信等）各种原因没去操作，只能在事后独自叹息。有没有解决的办法呢？今天咱们说一个不常说的词——容错能力。顾名思意，容易能力就是你容错度，你能容许自己最大犯多大的错误的能力。你把自己的交易体系看成一个系统，容错能力的目标就是即使某一步出现了错，也不影响系统整体运行。那么交易时如何才能提高我们的容错能力呢？一定要进行仓位管理，它是风险管理的第一步，运用到交易时就是你要尽可能的将一笔交易拆分成多笔，这样的好处是即使一笔出错，也可以通过其他几笔交易进行挽救。举个例子，在回调时，可能的回调位置分别在0.382、0.5和0.618附近，如果你想在回调时建仓，由于你不知道它能回调到哪里，就可以分别在以上点位附近挂单。如果继续细分，你也可以将0.382附近的单子再拆分，分别放在0.382上面、0.382处和0.382下方附近，这样做的好处是既分散了风险又提高了能买到概率。很多时候你觉得人家的点位很准，其实只是人家分批挂单的结果。总之，交易的大忌就是在关键点位重仓去赌，哪怕你再确信，也不要这样去做，那些遭受重大损失的人，很多都是在关键点下了重注造成的。
