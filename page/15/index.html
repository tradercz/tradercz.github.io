10月18日比特币行情：震荡后有向上突破的需求

发表于 2020-10-18 分类于 行情分析

过去24小时，整个市场的交易量594亿美金，比前一天减少了约200亿，现在是北京时间10月11日的上午09点03分，BTC即时价格11382USDT，下面是具体分析。

1）过去一天BTC都在11170-11400区间震荡，向上尝试突破11400未果，我们看这次下跌后的抄底情况，力度还是不小的，因此在不跌破11170的情况下就没有看空的理由，上方关注11400突破的情况，突破后就会进一步测试11500-11600压力区间。总结一下，周末交投清淡，行情小幅震荡，震荡后有向上突破的需求。

2）我经常被同学问到这样一个问题：老师，今天适合做多还是做空？其实我理解他的意思，那就是今天看多还是看空，但我却很难回答这个问题，主要有以下几个原因。一、很多时候看多并不一定就要做多，因为适合做多的点位只有在某一瞬间才会有，过后再做多，盈亏比就不高了。二、一天内的波动具有很强的随机性，虽然大小趋势可以判断，但短期涨跌不容易把握，有很多分析师都推荐大家逢低做多也正是这个原因，而同学们不理解“逢低”的意思，或者每个人对“逢低”的理解不同，都会造成操作的不同。三、并不是每天都适合操作，哪怕你做短线，很多时候也要等上几天到一周的时间才有适合的机会，等待是最重要的策略。四、每个人的操作周期不同，操作风格不同，生搬硬套的策略很容易出现错误的结果。最后再强调一下，一定不要拿着分析师的观点就往自己身上套，你的每一次操作都应该在你的交易框架中进行，别人的观点可以借鉴，但不要盲从，有同学听了某个分析师的建议，结果却与想象的不同，就开始将责任推到该分析师身上，这样也是不对的，我们应该有自己的思考。目前币圈很多的收费骗局并不是一方面造成的，除了有分析师的人品不好外，还有很大的一部分原因是散户只想吃现成的这种惰性把分析师惯坏了，要知道，雪崩时没有一片雪花是无辜的，而币圈的种种乱象，与我们每个人都有关系。

10月17日比特币行情：行情未破位

发表于 2020-10-17 分类于 行情分析

过去24小时，整个市场的交易量808亿美金，比前一天略多，现在是北京时间10月17日的上午09点17分，BTC即时价格11310USDT，下面是具体分析。

1）短期上涨的希望被OKEx的黑天鹅事件打破了，最低跌至11200点，但并没有跌破11172的近期低点，也就没有破位，目前仍在11170-11750区间震荡，这样看来，震荡后还是有机会继续上行的，只是OKEx事件的后续影响有多大，暂时还不清楚，且目前也没有进一步严重的迹象，我们就耐心等待吧。总结一下，黑天鹅后的修复震荡，行情没破位，一旦再次突破11400点，就可能走出一波短期上涨行情。

2）今天咱们做一个心理实验，假设现在有人用20000美金的价格（历史最高价）来购买你手中的BTC，但有一个要求，那就是2年之内你无法再进入币圈交易，你会卖给他吗？我们来分析一下这个实验，如果你同意卖给他，20000美金的价格一定是赚钱的，但同时你失去了未来两年BTC突破20000美金的机会，也就是你用确定性来换不确定性，乍一看是笔不错的交易。那么这个实验想说明的是什么呢？是两种思维方式，有的地方称穷人思维和富人思维，但我觉得这样称呼难免有洗脑的嫌疑，我们就直接称为A思维和B思维就好了，A思维要的是确定性，B思维要的是不确定性的机会。我们又说过，交易就是将不确定性尽量变成确定性，这样看A思维更符合交易的原则，但在世界上极为聪明的人中，很多人都是B思维，比如Facebook拒绝了雅虎10亿美金的收购，而Snapchat也拒绝了Facebook 30亿美金的收购，你现在看，这两次拒绝是多么的明智啊，但在当时却不这么认为，一面是确定性的收益，一面是有潜力但前途未卜的未来，如果不是极度聪明且有魄力的人，不会作出拒绝的决定。说到这里，我们就能引出一个概率权的概念了，而A思维，就倾向于拿到确定的东西而放弃概率权。B思维正好相反，每次选择的时候，都愿意根据成功的概率，和自己的本金多少来下注，这就是在计算期望值，真正算出自己的概率权。当然，B思维并不是要让你去赌，这只是一种思维方式，毕竟人生从来都不是只有二选一，你拥有更多的思维方式和思维习惯，就能更加容易的打开人生的局面。

2）最近有同学问美国大选可能对BTC走势造成的影响，我说一下自己的看法。我们知道，美国下一届总统将在特朗普和拜登之间产生，那么二者当选有什么不同呢？如果特朗普当选，会对经济提出更加激进的刺激政策，那么美元将走高，黄金受压，如果拜登当选，那么可能就不会太激进，黄金就可能走高。如果BTC继续跟随黄金，那么特朗普当选不利于BTC，拜登当选有利于BTC，请注意这里的前提，那就是BTC要跟随黄金才行，如果BTC不跟随黄金了，那这个规律就不灵了。另外，以上说的只是短期影响（比如1-2周时间），长期看，不管谁当选，都会继续提出刺激经济的政策，黄金温和上涨的逻辑不会变，BTC的大周期也在这，BTC也会逐步的走高。总之，大选可能对BTC短期会造成一定的波动，但长期一定是看涨的。还有，现在距离大选还有不到20天的时间，如果BTC跟随黄金，那么这段时间内BTC大概率还会继续震荡，暴涨暴跌都不太可能。当然，我很期待BTC能影响黄金而不是被黄金影响的那一天的到来。

2）今天咱们谈谈投资比特币需要满足什么样的基础条件，注意我说的是投资而不是投机。所谓金融领域的投资，就是投入一定的金钱和时间来获取利益的最大化。这里面有两个条件，一个是金钱，一个是时间。金钱大家都知道，也就是你的初始本金，或多或少你都是有的，那么时间呢？并不是每个人都有的。有人说谁没有时间呢？我告诉你，你投入一笔资金在BTC上，为的是将来获取收益最大化，结果遇到了BTC大跌，于是你慌忙卖出了，那么对于这次投资而言，你就没有了时间，也就是你放弃了耐心等待而选择了止损。需要注意的是，这里不讨论卖出的操作对不对，其实也没有对不对之分，就看你是选择用时间换金钱，还是用金钱换时间了（前参考我们之前说过的时间换空间与空间换时间的概念），所以，要想做一次好的投资，一要金钱，二要时间，其中时间更重要些。咱们再延伸一下，说说投机，我认为的投机就是短线交易，它并不是贬义词，与投资一样，投机也非常重要，你只有克制住了自己的贪念，战胜了人性弱点才能做好投机。说白了，投资与投机其实就是长线交易和短线交易这两种思维方式，不会投机的人很可能会陷入到牛市陷阱中出不来，不会投资的人虽然每天都在操作财富却一直在缩水，因此，对于我们每个人来说，这两种思维同等重要，而如何让这两种思维达成平衡，才是我们每个人应该考虑的问题。

2）有同学问“站稳”的概念，今天咱就来谈一谈。所谓站稳，在技术分析里并没有准确的概念，有人会将突破并经过一次有效回踩称为站稳，有人会将突破后运行一段时间没有跌回来视为站稳，有人会将突破某一百分比视为站稳（比如突破某个关键压力位3%），也有人会综合各方面进行判断……我个人通常会将突破回踩和突破运行一段时间结合着来判断。这里需要注意几个问题，一是判断是否“站稳”要考虑周期，比如突破5分钟压力位，你就不能用日线当做周期，相应的突破日线压力位，也不能用5分钟作为周期。二是“站稳”这个词具有时效性，也就是只对当前时间点有效，现在看起来站稳了，也不能保证以后就不会再跌回来了。三是“站稳”具有较强的主观性，它在一个人的交易系统里称为站稳，但在另一个人的交易系统里就可能不是站稳了。至此，你是不是觉得我还是没有将“站稳”解释清楚？是的，由于没有具体的可量化的数据，就很难解释清楚什么才是“站稳”。以当前的行情为例，BTC已经突破9月19日高点并进入更高的箱体，你可以说它站稳了（已经运行了一段时间），也可以说没有站稳（没有经过有效回踩，注意，现实中完美回踩的情况并不多）。看到这里，你是不是有被耍了的感觉？不要急，因为站稳是否根本不重要，重要的是发生突破信号后你要做相应的操作，比如突破加仓或者（做空的）突破减仓，这时候一定不要怕后市再跌回来，跌回来你止损就行了呗，该大胆的时候一定要大胆。很多时候，我们都将时间浪费在了不重要的事情上面，从而忽略了真正重要的事情。总结一下：一，站稳是否没有准确的概念，没有可量化的数据，你经验越丰富，越能增加判断的准确性。二，与其花时间判断是否站稳，不如根据信号严格执行。

10月10日比特币行情：多头离成功最近的一次

发表于 2020-10-10 分类于 行情分析

过去24小时，整个市场的交易量938亿美金，比前一天略多，现在是北京时间10月10日的上午09点04分，BTC即时价格11057USDT，下面是具体分析。

1）35天后，BTC再次站上了60日线，这是中期看涨的主要指标之一。现在面临两个问题，一是并未站稳60日线，二是还没有突破9月19日的高点11180，那么后市就可能继续在60日线附近震荡两三天，一旦突破11180点，则可认为中期上涨趋势正式形成。对于当前市场的看法：目前多头趋势并未形成，后市要保持上涨趋势，最好不能跌破10800点（注意刺破是可以的）。总结一下，这是近期多头离成功最近的一次，而且现在看来是很有希望的。

2）今天我们谈论的主题是完美主义，完美主义的意思就不说了，直接进入正题。在交易中，完美主义主要有以下几种表现：总想买在最低点，总想卖在最高点，总想着一下子就干一票大的，总想着暴富，看不起“蝇头小利”，拒绝接受失败的交易，将失败的原因归结在其他方面，坚持自己是完美的、没有错误的。其实这个话题到这里就可以结束了，毕竟我的读者都是相当聪明的人，看到问题就能想到解决办法，然而交易难就难在它是反人性的，很多时候你明知自己的弱点却无法做出改变，那我就再深入一步。完美主义者总想证明自己聪明、正确、高瞻远瞩，绝不轻举妄动，其本质就是害怕失败且拒接接受自己失败，从而无法升级自己。如果你有完美主义情节，那么一定要试着接受自己的平庸，丢掉暴富的想法，从“蝇头小利”做起，接受不完美的交易。巴菲特说过这样一句话：“在投资方面我们之所以做得非常成功，是因为我们全神贯注于寻找我们可以轻松跨越的1英尺高的栏杆，而避开那些我们没有能力跨越的7英尺高的栏杆”。然而币圈的现状是，很多没有能力跨过7英尺高栏杆的人却总忙着在找7英尺的栏杆，这也是完美主义的错用。这里我想问一句，即使找到了7英尺高的栏杆，你确信你有足够的能力和运气跳过去吗？

10月9日比特币行情：尊重盘面，短期看多

发表于 2020-10-09 分类于 行情分析

过去24小时，整个市场的交易量911亿美金，比前一天增加了100多亿，现在是北京时间10月9日的上午08点54分，BTC即时价格10880USDT，下面是具体分析。

1）BTC于昨晚9点选择了向上突破，并且成功突破了下降趋势线，完成了一次漂亮的日线级别的突破，后市怎么走？既然向上突破了，那就继续看多，接下来的目标是11000（60日线附近）以及11179（9月19日高点），如果能顺利突破这两个目标位，那么后市成功上攻12468并创下新高就指日可待了。不过这里需要注意两点，一是消息面刺激是昨晚拉升的主要原因，一般的消息市可持续性都不强，二是目前仍受60日线的压制，且60日线已经掉头向下，这种情况下可以看多但不要太乐观。总结一下，尊重盘面，短期看多，看最终能走到哪里吧。

2）昨天的放量拉升并未给多头带来过多的喜悦，这要是放在拉升的末端，多头一定会非常的兴奋，他们会认为行情还会继续上涨，可这次拉升却发生在长时间的下跌后，有很多人都在怀疑这波拉升的真实性，还有的在想后面会不会来个画门行情，这里我说说我的看法吧。一、我现在看不出来后市会不会画门，但即使画门我也不会意外，毕竟这种行情在今年已经发生太多次了，二、行情总是在怀疑中前行的，这里怀疑的人越多，越有可能真的向上突破。三、如果你对当前行情没有头绪，那就只看盘面，不考虑其他因素（奥卡姆剃刀，盘面上是向上突破），假如后面画门的话，我们仍旧尊重盘面就好。
